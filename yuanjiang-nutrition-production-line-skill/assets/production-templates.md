## 7. 审稿清单

### 事实安全

- [ ] 每个强结论都有来源；
- [ ] 没有编造指南 / 论文 / 年份 / 机构；
- [ ] 没有把网页伪装成论文；
- [ ] 没有把单篇论文说成定论；
- [ ] 没有把安全上限说成推荐摄入量；
- [ ] 不确定内容已标注、追问或删除。

### 表达安全

- [ ] 不制造恐慌；
- [ ] 不羞辱体型、饮食习惯或疾病状态；
- [ ] 不说“一定”“必须”“彻底避免”等绝对话，除非标准明确禁止；
- [ ] 不营销产品；
- [ ] 不推荐品牌、保健品、减肥药、药膳剂量。

### 媒体安全

- [ ] 图片/视频没有长段乱码中文；
- [ ] 字幕与画面分离；
- [ ] 没有水印、伪 logo、伪引用；
- [ ] 文件真实存在；
- [ ] prompt 与生成记录已保存。
