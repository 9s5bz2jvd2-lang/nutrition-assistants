# 圆酱营养科普生产线

这是一个可迁移的 LingTai custom skill，用于把营养学、食品安全、国家标准、医学指南、论文、官方发布或新闻资料，生产成「圆酱说营养」风格的科普内容包。

它支持两种运行方式：

1. **单体 Agent 工作流**：资料 → 证据 → 大白话 → 海报/视频/音频脚本 → 审稿清单。
2. **灵台分身协作工作流**：总编主控 + 资料获取 + 证据提取 + 营养审稿 + 文案 + 图片 + 视频 + 音频 + 素材包整理。

媒体产出规则：

- 有图片/视频/音频工具时，可以直接生成文件；
- 没有工具时，不假装生成，而是输出 prompt、分镜、字幕、口播稿和素材清单；
- 图片/视频不直接生成大段中文，中文后期排版或字幕独立输出。

核心红线：

- 不编造指南、论文、年份、机构、数据或专家共识；
- 不把网页科普伪装成论文；
- 不把单篇研究夸大成定论；
- 不制造恐慌、不羞辱体型、不营销产品；
- 个体化医疗、用药、化验单、孕妇婴幼儿严重慢病等高风险场景转专业人士。

给 agent 使用时，请读 `SKILL.md`；迁移说明见 `assets/handoff.md`。
