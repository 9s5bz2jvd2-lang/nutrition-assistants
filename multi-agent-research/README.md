# 营养学 | 多智能体研究（Multi-Agent Research）

> 状态：**进行中 / In Progress** — 本目录目前为占位说明，尚无可用的完成功能。

## 这是什么？

这是 Nutrition Assistants 合集中规划中的研究方向：探索用**多 Agent 协作**的方式，搭建一条营养学文献 / 证据 / 科普内容的生产线。

设想中的分工（尚在设计、未实现）：

- **检索 Agent**：从公开指南、文献中收集营养学相关资料；
- **证据评估 Agent**：对资料做证据分级与一致性核查；
- **蒸馏 Agent**：把指南/文献结构化为知识库（KPK 等）；
- **科普写作 Agent**：把结构化知识转写为面向大众的科普表达；
- **审校 Agent**：做事实核查与安全边界把关。

## 安全与边界

本研究方向面向**教育科普与专业辅助**，不构成医疗诊断或治疗建议，不替代医生或营养师的临床判断。后续产出会延续合集统一的循证（evidence-based）与安全边界原则。
