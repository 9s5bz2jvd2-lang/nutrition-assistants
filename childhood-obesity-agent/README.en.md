# Childhood Obesity Dietary-Nutrition Assistant Agent

English version translated from the existing Chinese README.

> **Copy-and-use · Full System Prompt Agent**

A complete AI Agent system prompt distilled from the National Health Commission's *Dietary and Nutrition Guide for Children and Adolescents with Obesity (2024 Edition)*.

## What is this?

This repository provides a **complete System Prompt**. Copy all content into the system-prompt / persona field of any AI platform (such as ChatGPT, Coze, ERNIE Bot, Tongyi Qianwen, Doubao, etc.) to get a professional childhood-obesity dietary-nutrition assistant.

## Content Scale

| Module | Content |
|------|---------|
| Role definition | Identity, style, and language switching |
| Key data | BMI cutoffs, activity, sleep, food intake, and energy |
| TCM syndrome differentiation | 5 syndrome patterns plus food-medicine recommendations |
| Dietary formulas | 10 complete formulas with ingredients, preparation, and usage |
| Food choices | 9 categories: preferred / limited / not recommended |
| Menu quick reference | 5 regions × 24 days of complete menus |
| Scenario responses | Strategies for 6 common scenarios |
| Common misconceptions | Corrections for 10 major misconceptions |
| Safety red lines | Anti-patterns and disclaimer |

## Files

- `agent_prompt.md` — Complete System Prompt, ready to paste and use.
- `README.md` — Chinese README.

## How to Use

1. Open `agent_prompt.md`.
2. Copy all content.
3. Paste it into the target AI platform's System Prompt / persona input field.
4. Save and start chatting.
