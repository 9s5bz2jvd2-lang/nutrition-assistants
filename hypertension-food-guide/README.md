## 指南出处

- **全称**：《成人高血压食养指南（2023年版）》
- **发布单位**：国家卫生健康委办公厅
- **制定依据**：《健康中国行动（2019—2030年）》《国民营养计划（2017—2030年）》

## 功能特点

- **风险评估**：血压水平自评（3级分级）+心血管风险分层
- **食养原则**：5条官方原则——减钠增钾为核心，覆盖膳食/运动/体重/生活方式全方位干预
- **三种膳食模式**：DASH饮食、东方健康膳食模式、CHH-Diet（中国心脏健康膳食）
- **中医辨证**：7种证型（肝火上炎/痰湿内阻/瘀血内阻/阴虚阳亢/肾精不足/气血两虚/冲任失调）
- **食养方库**：14个食养方（粥6+茶饮4+汤3+饭1），按证型分类选用
- **地区适配**：7大地区×4季节×3天 = **84套食谱**完整收录
- **食物选择**：5大类推荐量 + 宜选/限制食物清单 + 隐形钠识别
- **科普风格**：说人话、给具象量、指误区——精准但不端着
