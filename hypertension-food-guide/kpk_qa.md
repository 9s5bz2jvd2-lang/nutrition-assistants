# KPK 知识点：指南问答版（KPK-12）

## KPK-12：指南问答版——10个官方问答精要

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-12 |
| **核心知识** | 《成人高血压食养指南（2023年版）》官方问答，涵盖制定目的、依据、原则、使用方法和注意事项 |
| **适用人群** | 成人高血压患者、基层卫生工作者、营养指导人员 |
| **食养建议** | **Q1 制定目的和意义是什么？** 满足人民健康需求，为预防和控制高血压、改善膳食、提高营养健康水平提供科学指导。意义：①发挥现代营养学和传统食养中西医优势；②针对不同证型/地区/季节编制食谱提升个性化可操作性；③推动中西医结合传承传统食养文化。**Q2 制定依据是什么？** 《健康中国行动（2019—2030年）》（鼓励发展传统食养服务）、《国民营养计划（2017—2030年）》（发挥中医药特色优势制定食养指南开展慢性病食养指导）。**Q3 制定四原则？** ①科学性（以营养科学/中医理论/循证文献为依据）；②协调一致性（与现行法律法规协调）；③公开透明（广泛征求多方面意见）；④指导性与实用性结合（结合不同证型/地区/季节列出食谱示例）。**Q4 国内外参考标准？** 《中国居民膳食指南(2022)》《高血压患者膳食指导》(WS/T430-2013)《中国高血压防治指南(2018年修订版)》《高血压中医诊疗指南》《中国人群身体活动指南(2021)》《国际高血压学会(ISH)高血压管理声明》(WHO2003)《预防心血管疾病风险评估和管理指南》(WHO2007)《2007动脉高血压管理指南》等。**Q5 主要内容？** 食养原则和建议+附录（食物选择、证型食药物质及食养方、地区季节食谱、食物交换表）。每地区每季至少3天食谱（含食药物质）。**Q6 发布后如何使用？** 面向成人高血压患者+基层卫生工作者(含营养指导人员)。不适用于病情复杂合并多种疾病者。不可替代药物治疗。**Q7 膳食上注意什么？** 全面均衡多样。"减钠增钾饮食清淡"：减少食盐含钠调味品加工食品摄入→增加富钾食物(蔬果豆类)；主食增加全谷物薯类(全谷物杂豆占谷类1/4～1/2)；动物性食物选鱼虾禽瘦肉每天120～200g+奶类300～500g；限制高脂高胆固醇食品(油炸动物内脏)；不饮或限酒不喝少喝含糖饮料；选不饱和脂肪酸植物油交替使用；针对不同证型选食药物质和方案。**Q8 选择食药物质注意什么？** 首先准确辨证→"因人因时因地"三因原则选相应食药物质→将所提供能量宏量营养素纳入全天饮食总量→已知过敏者禁用→正在服药可能有禁忌者需咨询医师/药师/营养指导人员。**Q9 参考食谱注意什么？** 六点：①调整食量量体裁衣(结合地区季节证型年龄性别特点调整)；②同类互换享受美食(按食物交换表互换注重当地食材和文化)；③少盐少油循序渐进(逐步降低培养清淡口味)；④少量多餐提倡分餐(合理分配各餐次能量个性化)；⑤吃动平衡健康体重(适度运动能量平衡)；⑥养成习惯贵在坚持(持续培养健康饮食习惯) |
| **食材/方剂** | 无新增食材，为使用指南说明 |
| **注意事项** | 本指南为食养指导不可替代药物治疗；病情复杂合并多种疾病者需在专业人员指导下个体化设计 |
