# 高血压食养助手 —— KPK知识库（整合版）

> 基于《成人高血压食养指南（2023年版）》| 国家卫生健康委办公厅发布 | 共12个KPK知识点

## KPK-01：减钠增钾，饮食清淡

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-01 |
| **核心知识** | 钠盐摄入过多可增加高血压风险。增加膳食中钾摄入量可降低血压。所有高血压患者均应限制来源于各类食物的钠盐摄入 |
| **适用人群** | 成人高血压人群 |
| **食养建议** | 每人每日食盐摄入量逐步降至**5g以下**；增加富钾食物（新鲜蔬菜、水果、豆类）摄入。我国居民膳食中**75%以上的钠来自家庭烹调盐**，其次为高盐调味品；加工食品也是重要钠盐来源。清淡饮食，少吃含高脂肪、高胆固醇的食物。肾功能良好者可选择**高钾低钠盐**；不建议服用钾补充剂；肾功能不全者补钾前应咨询医生。适当选择富含钙、镁的食物。限制膳食脂肪和胆固醇（油炸食品、动物内脏），少吃加工红肉制品（培根、香肠、腊肠等）|
| **食材/方剂** | 高钾食物：新鲜蔬果、豆类、菌藻类、薯类；高钠需限制：咸菜、酱类、味精/鸡精/蚝油、加工肉制品、方便面/挂面 |
| **注意事项** | 减盐要循序渐进；注意隐形钠（酱油含钠约5-7g/100ml）；烹调时可用醋/柠檬汁/香料替代部分盐 |
| **关联KPK** | KPK-02（合理膳食），KPK-08（食物选择）|

## KPK-02：合理膳食，科学食养

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-02 |
| **核心知识** | 平衡膳食应由五大类食物组成，高血压患者应在平衡膳食基础上调整优化食物种类和重量，遵循"辨证施膳""辨体施膳"原则 |
| **适用人群** | 成人高血压人群 |
| **食养建议** | 五大类食物：（1）谷类200～300g/天（全谷物和杂豆50～150g），薯类50～100g/天；（2）深色蔬菜占总蔬菜量一半以上；（3）动物性食物120～200g/天；（4）奶类300～500g/天；（5）限制添加糖和含钠调味品。7种证型辨证施膳：肝火上炎→清淡平肝潜阳；痰湿内阻→易消化健脾运湿；瘀血内阻→温平活血通络；阴虚阳亢→养阴滋阴潜阳；肾精不足→温补补益肝肾；气血两虚→细软滋补补益气血；冲任失调→富营养调和冲任 |
| **食材/方剂** | 全谷物（糙米/燕麦/荞麦等）；新鲜蔬菜≥300g/天；新鲜水果200～350g/天；脱脂/低脂奶；豆制品 |
| **注意事项** | 蔬菜水果不可相互替代；少选加钠盐谷类制品（咸味面包/方便面/挂面）；推荐不同种类植物油交替使用 |
| **关联KPK** | KPK-01（减钠增钾），KPK-03（吃动平衡），KPK-07（中医分型）|

## KPK-03：吃动平衡，健康体重

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-03 |
| **核心知识** | 超重和肥胖是高血压患病的重要危险因素，尤其是中心性肥胖。推荐将体重维持在健康范围内 |
| **适用人群** | 成人高血压人群（尤其超重/肥胖者）|
| **食养建议** | BMI **18.5～23.9kg/m²**（65岁以上可适当放宽）；男性腰围**＜85cm**，女性**＜80cm**。超重肥胖者减重：能量比原来减少**300～500kcal/天**。18～49岁成年人轻活动能量需要量：男**2250kcal**，女**1800kcal**。运动：每周累计2.5～5小时中等强度有氧活动；或每周4～7天每天30～60分钟中等强度活动。运动可改善血压水平 |
| **食材/方剂** | 控制能量密度：减少油炸/甜点/含糖饮料；增加全谷物蔬菜增强饱腹感 |
| **注意事项** | 减重速度不宜过快（0.5~1kg/周）；避免极端节食；BMI<18.5不适用减重 |
| **关联KPK** | KPK-02（合理膳食），KPK-05（自我管理）|

## KPK-04：戒烟限酒，心理平衡

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-04 |
| **核心知识** | 戒烟可降低心血管风险。过量饮酒显著增加高血压发病风险且随饮酒量增加而增加。精神紧张激活交感神经使血压升高 |
| **适用人群** | 成人高血压人群 |
| **食养建议** | 彻底戒烟避免被动吸烟。即使少量饮酒也有不良影响——建议**不饮酒**，饮酒者尽量戒酒。减轻精神压力进行压力管理（认知行为干预）；规律作息保证充足睡眠不熬夜 |
| **食材/方剂** | 无特定食材（生活方式干预）|
| **注意事项** | 突然戒酒可能出现戒断反应需医师指导下逐步减量 |
| **关联KPK** | KPK-05（监测血压）|

## KPK-05：监测血压，自我管理

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-05 |
| **核心知识** | 定期监测血压了解达标状态，遵医嘱生活方式干预+长期治疗。可选择三种推荐膳食模式之一 |
| **适用人群** | 成人高血压人群 |
| **食养建议** | 根据心血管总体风险及血压水平随诊。（1）**DASH饮食**——富含蔬果低脂乳禽鱼大豆坚果全谷物，限制含糖饮料红肉，低饱和脂肪酸和胆固醇，富钾镁钙；（2）**东方健康膳食模式**——东南沿海代表模式，清淡少盐食物多样谷物为主蔬果充足鱼虾丰富奶豆丰富身体活动量大；（3）**CHH-Diet**——较普通城市膳食减少钠和脂肪，增加蛋白碳水钾镁钙膳食纤维 |
| **食材/方剂** | DASH：燕麦菠菜香蕉脱脂奶三文鱼杏仁豆类；CHH-Diet：全谷物瘦禽肉鱼类大豆大量蔬果 |
| **注意事项** | 家庭自测固定时间（早晚各1次）、固定体位（坐位休息5分钟后测）；记录血压日记 |
| **关联KPK** | KPK-03（吃动平衡），KPK-06（疾病定义），KPK-09（血压分级）|

## KPK-06：高血压的定义、分类与流行病学

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-06 |
| **核心知识** | 高血压是以血压升高为主要特点的全身性疾病，是我国当前面临的重要公共卫生问题 |
| **适用人群** | 成年人（尤其高危人群筛查）|
| **食养建议** | 定义：未用降压药情况下非同日3次诊室血压SBP**≥140mmHg**和/或DBP**≥90mmHg**。SBP≥140且DBP<90为单纯收缩期高血压。既往高血压史用药中即使<140/90仍诊断高血压。流行病学（《中国居民营养与慢性病状况报告2020》）：18+居民患病率**27.5%**（18-44岁13.5%，45-59岁37.8%，60+岁59.2%）；估计**2.45亿**患者。2017年高SBP导致**254万人死亡**（95.7%CV死亡）。治疗可年减**80.3万例CV事件**获**120万QALY**。危险因素：高钠低钾膳食、过量饮酒、超重肥胖(尤其中心性)、缺乏运动 |
| **食材/方剂** | 无 |
| **注意事项** | 诊室血压由医护标准条件测量是诊断常用方法；家庭自测可用作辅助但需校准设备 |
| **关联KPK** | KPK-07（中医分型），KPK-09（血压分级）|

## KPK-07：中医辨证分型——7种证型的食养思路

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-07 |
| **核心知识** | 高血压属中医学"眩晕""头痛"范畴，主要病位在心肝脾肾，病性有实有虚以**虚实夹杂**为主，共7种证型 |
| **适用人群** | 成人高血压人群（需辨证后选用对应食养方案）|
| **食养建议** | （1）**肝火上炎证**——头晕胀痛/面红目赤/烦躁易怒；耳鸣如潮/胁痛口苦/便秘溲黄；舌红苔黄脉弦数 → 清淡平肝潜阳。（2）**痰湿内阻证**——头重如裹；胸脘痞闷/纳呆恶心/呕吐痰涎/身重困倦/少食多寐；苔腻脉滑 → 易消化少餐健脾运湿。（3）**瘀血内阻证**——头痛如刺痛有定处；胸闷心悸/手足麻木/夜间尤甚；舌暗脉弦涩 → 温平活血通络。（4）**阴虚阳亢证**——眩晕耳鸣/腰酸膝软/五心烦热；头重脚轻/口燥咽干/两目干涩；舌红少苔脉细数 → 养阴滋阴潜阳。（5）**肾精不足证**——心烦不寐/耳鸣腰酸；心悸健忘/失眠梦遗/口干口渴；舌红脉细数 → 温补补益肝肾。（6）**气血两虚证**——眩晕时作/短气乏力/口干心烦；面白自汗盗汗/心悸失眠/纳呆腹胀便溏；舌淡脉细 → 细软滋补补益气血。（7）**冲任失调证**——更年期前后头痛头晕；心烦失眠/胁痛全身不适；血压波动；舌淡脉弦细 → 富营养调和冲任 |
| **食材/方剂** | 详见附录4各证型食养方举例（每证2方共14个）|
| **注意事项** | 辨证应由专业中医师进行；同一患者可能兼夹多型；证型可能随病程变化定期复评 |
| **关联KPK** | KPK-02（辨证施膳），KPK-06（疾病定义），KPK-11（食养方）|

## KPK-08：成人高血压患者的食物选择（附录1）

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-08 |
| **核心知识** | 高血压患者在五大类框架下选低钠高钾富含钙镁食物，限制高盐高脂高胆固醇食物 |
| **适用人群** | 成人高血压人群 |
| **食养建议** | （一）谷薯类：谷类200~300g/天（全谷物杂豆**50~150g**），薯类50~100g/天；少选加钠盐制品。（二）动物性：鱼禽蛋瘦肉120~200g/天；奶类300~500g/天。（三）大豆坚果：25~35g/天。（四）蔬果：总量**>500g/天**（蔬菜≥300g，水果200~350g）；深色蔬菜占**一半以上**。（五）油脂：不饱和脂肪酸植物油20~25g/天交替使用。（六）调味：食盐≤5g/天，减含钠调味品 |
| **食材/方剂** | **宜选**：全谷物、深色蔬菜、新鲜水果、鱼虾、去皮禽肉、低脂奶、豆制品、坚果少量。**限制**：咸菜酱菜腐乳、加工肉类、油炸食品、动物内脏、方便面挂面咸面包、高盐调味品 |
| **注意事项** | 蔬果不可互相替代；天然香料替代部分盐增风味 |
| **关联KPK** | KPK-01（减钠增钾），KPK-02（合理膳食）|

## KPK-09：高血压水平分级与心血管风险分层

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-09 |
| **核心知识** | 高血压按血压水平分3级，结合其他危险因素风险分层（低危/中危/高危/很高危）指导管理强度 |
| **适用人群** | 成人高血压人群及基层卫生工作者 |
| **食养建议** | **血压分类**：正常<120/<80；正常高值120-139/80-89；高血压≥140/≥90；1级140-159/90-99；2级160-179/100-109；3级≥180/≥110；单纯收缩期≥140/<90。注：SBP/DBP分属不同级别时以较高为准。**风险分层**：无其他危险因素时SBP130-139/DBP85-89→低危；140-159/90-99→中危；160-179/100-109→高危；≥180/≥110→很高危。每增加一层危险因素提升一级风险。临床并发症/CKD≥4期/并发症糖尿病均为高/很高危 |
| **食材/方剂** | 无特定食材（分层决定干预力度）|
| **注意事项** | CKD=慢性肾脏疾病；靶器官损害包括左室肥厚蛋白尿视网膜病变等 |
| **关联KPK** | KPK-06（定义流行病学），KPK-05（监测管理）|

## KPK-11：不同证型食养方举例（附录4，14个食养方）

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-11 |
| **核心知识** | 针对每种中医证型提供2个具体食养方（粥/汤/饮/膳），共14个供参考选用 |

## KPK-12：指南问答版——10个官方问答精要

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-12 |

| **食养建议** | Q1目的意义：预防控制高血压改善膳食提高营养健康水平；发挥中西医优势；针对不同证型地区季节编制食谱提升个性化可操作性；推动中西医结合传承传统食养文化。Q2依据：《健康中国行动2019-2030》《国民营养计划2017-2030》。Q3四原则：科学性/协调一致性/公开透明/指导性与实用性结合。Q4参考标准：《中国居民膳食指南2022》《高血压患者膳食指导》WS/T430-2013、《中国高血压防治指南2018修订版》《高血压中医诊疗指南》《中国人群身体活动指南2021》、ISH声明WHO2003/2007等。Q5主要内容：食养原则建议+附录（食物选择/证型食药物质及食养方/地区季节食谱/食物交换表）。Q6使用对象：成人高血压患者+基层卫生工作者；不适用于复杂合并症；不可替代药物。Q7膳食注意：减钠增钾清淡饮食；全谷物占主食1/4~1/2；动物性120~200g+奶300~500g/天；限制高脂高胆固醇；不饮限酒不喝含糖饮料；不饱和脂肪酸植物油交替；辨证选食药物质。Q8食药物质注意：准确辨证→三因原则→纳入全天总量→过敏禁用→药忌咨询专业。Q9食谱注意六点：①调食量量体裁衣②同类互换享受美食③少盐少油循序渐进④少量多餐提倡分餐⑤吃动平衡健康体重⑥养成习惯贵在坚持 |
