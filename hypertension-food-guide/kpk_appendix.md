# KPK 知识点：附录知识

## KPK-08：成人高血压患者的食物选择（附录1）

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-08 |
| **核心知识** | 高血压患者应在五大类食物框架下，选择低钠、高钾、富含钙镁的食物，限制高盐高脂高胆固醇食物 |
| **适用人群** | 成人高血压人群 |
| **食养建议** | **（一）谷类和薯类**：增加全谷物和薯类摄入，粗细搭配。推荐谷类200～300g/天（全谷物和杂豆**50～150g**），薯类50～100g/天；少选或不选加入钠盐的谷类制品（咸味面包、方便面、挂面等）。**（二）动物性食物**：选择鱼禽蛋瘦肉，平均每天120～200g；少选或不选高盐高脂高胆固醇动物性食物；推荐各种奶制品300～500g/天。**（三）大豆和坚果**：每日大豆及坚果共25～35g。**（四）蔬菜水果**：新鲜蔬果摄入总量应**>500g/天**（蔬菜≥300g，水果200～350g）；深色蔬菜占蔬菜总量的**一半以上**。**（五）油脂**：选用含不饱和脂肪酸的植物油，推荐不同种类交替使用，烹调油20～25g/天。**（六）调味品**：食盐≤5g/天，减少酱油、酱类、蚝油、鸡精、味精等含钠调味品 |
| **食材/方剂** | **宜选**：糙米/燕麦/荞麦/玉米等全谷物；深色蔬菜（菠菜/油菜/西兰花等）；新鲜水果；鱼虾类；去皮禽肉；脱脂/低脂奶；豆制品（豆腐/豆浆/豆腐干）；坚果（核桃/杏仁少量）。**限制**：咸菜/酱菜/腐乳；加工肉制品（培根/香肠/腊肠）；油炸食品；动物内脏；方便面/挂面；咸味面包；高盐调味品 |
| **注意事项** | 蔬菜和水果不可相互替代（营养素构成不同）；烹调时可用天然香料（葱/姜/蒜/醋/柠檬/胡椒等）替代部分盐以增强风味 |
| **关联KPK** | KPK-01（减钠增钾），KPK-02（合理膳食） |

## KPK-10：不同证型食药物质及新食品原料推荐（附录1表1.2）

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-10 |
| **核心知识** | 7种证型各有对应的推荐食药物质，需辨证后选用。食药物质来源包括：①食药物质目录品种；②新食品原料；③普通食品原料（药食两用属性） |
| **适用人群** | 成人高血压人群（需准确辨证） |
| **食养建议** | **7证型食养方向与常用食材**：（1）**肝火上炎证**——平肝潜阳：菊花*、决明子*、槐米*、山楂*、芹菜。（2）**痰湿内阻证**——健脾运湿：橘皮*、薏苡仁*、茯苓*、白扁豆*、荸荠。（3）**瘀血内阻证**——活血通络：山楂*、当归*、黄芪*（限执业医师）、三七*。（4）**阴虚阳亢证**——滋阴潜阳：百合*、银耳、枸杞子*、苦丁茶*、菊花*、决明子*。（5）**肾精不足证**——补益肝肾：枸杞子*、桑椹*、黑芝麻*、黄精*、山药*。（6）**气血两虚证**——补益气血：黄芪*（限执业医师）、当归*、大枣*、龙眼肉*、莲子*、百合*。（7）**冲任失调证**——调和冲任：当归*、阿胶、大枣*、玫瑰花*。注：*标记为食药物质或需在专业人员指导下使用 |
| **食材/方剂** | 各证型具体食养方详见附录4（每证2方共14个） |
| **注意事项** | "因人、因时、因地"三因原则选择食药物质；将所提供的能量与宏量营养素纳入全天饮食总量考虑；已知过敏者禁用；正在服用药物可能与食药物质有禁忌者须咨询专业医师/药师/营养指导人员 |
| **关联KPK** | KPK-07（7种中医分型），KPK-11（食养方举例） |

## KPK-11：不同证型食养方举例（附录4）

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-11 |
| **核心知识** | 针对每种中医证型提供2个具体食养方（粥/汤/饮/膳），共14个食养方供参考选用 |
| **适用人群** | 成人高血压人群（按证型选用） |
| **食养建议** | **肝火上炎证（2方）**：（1）**芹菜粳米瘦肉粥**——芹菜50g+粳米50g+枸杞子10g+猪瘦肉20g，切碎同煮为粥佐餐；（2）**枸杞菊花决明饮**——枸杞子10g+菊花10g+炒决明子15g，同煮去渣取汁调冰糖代茶饮。**痰湿内阻证（2方）**：（1）**橘皮饭**——橘皮10g+粳米100g，剪条同煮饭佐餐；（2）**雪羹汤**——荸荠+海蜇头各100g一同煮汤佐餐。**瘀血内阻证（2方）**：（1）**山楂粥**——山楂20g+粳米100g，煮水取汁熬粥加少量红糖佐餐；（2）**归芪蒸鸡**——鸡1只+黄芪30g+当归30g塞入腹内隔水蒸至熟烂分次食用（黄芪限执业医师）。**阴虚阳亢证（2方）**：（1）**苦丁菊花茶**——苦丁茶15g+菊花开水冲泡加冰糖代茶饮；（2）**百合银耳雪梨羹**——百合10g+银耳10g+雪梨25g+枸杞子5g加冰糖小火熬羹佐餐。**肾精不足证（2方）**：（1）**乌鸡菌汤**——乌鸡1只+木耳香菇各30g+枸杞子15g小火煮1小时熬至浓汤分次食用；（2）**桑椹粥**——桑椹15g+粳米50g同煮成粥佐餐。**气血两虚证（2方）**：（1）**莲子百合枣羹**——莲子15g+百合12g+大枣5枚+糯米50g加水煮成粥加少量冰糖佐餐；（2）**黄芪当归鲫鱼汤**——黄芪15g+当归10g+鲫鱼100g+生姜3片煮至熟烂加盐佐餐（黄芪限执业医师）。**冲任失调证（2方）**：（1）**当归羊肉萝卜汤**——当归10g+羊肉50g+白萝卜100g+生姜3片小火熬汤佐餐；（2）**阿胶鸡蛋汤**——鸡蛋1个+阿胶6g清水化开后打蛋搅拌加热成蛋花汤加盐佐餐 |
| **食材/方剂** | 共14个食养方（7证型×2方）：粥类6个、茶饮类4个、汤类3个、饭类1个 |
| **注意事项** | 标注"限执业医师"的药材（如黄芪、党参等）非试点地区仅限执业医师指导使用；食养方可根据个人口味适当调整但不建议大幅改动主料比例 |
| **关联KPK** | KPK-07（7种中医分型），KPK-10（食药物质推荐） |
