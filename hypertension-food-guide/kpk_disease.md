# KPK 知识点：疾病背景

## KPK-06：高血压的定义、分类与流行病学

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-06 |
| **核心知识** | 高血压是以血压升高为主要特点的全身性疾病，是我国当前面临的重要公共卫生问题 |
| **适用人群** | 成年人（尤其高危人群筛查） |
| **食养建议** | **定义**：在未使用降压药物情况下，非同日3次测量诊室血压，收缩压（SBP）**≥140mmHg**和（或）舒张压（DBP）**≥90mmHg**。SBP≥140且DBP＜90为**单纯收缩期高血压**。既往有高血压史、正在使用降压药者即使血压<140/90仍应诊断为高血压。**流行病学数据**（《中国居民营养与慢性病状况报告2020》）：18岁及以上居民高血压患病率**27.5%**；其中18～44岁患病率13.5%，45～59岁37.8%，60岁及以上59.2%；目前成人高血压患病人数估计为**2.45亿**。2017年高收缩压导致我国**254万人死亡**，其中95.7%死于心血管疾病。若对高血压患者进行治疗，每年可减少**80.3万例心血管事件**，获得**120万质量调整生命年（QALY）**。**危险因素**：高钠低钾膳食、过量饮酒、超重/肥胖（尤其是中心性肥胖）、缺乏身体活动 |
| **食材/方剂** | 无 |
| **注意事项** | 诊室血压：由医护人员在标准条件下按统一规范测量，是目前诊断高血压和观察疗效的常用方法。家庭自测血压可作为辅助参考但需使用校准设备 |
| **关联KPK** | KPK-07（中医分型），KPK-09（血压分级） |

## KPK-07：中医辨证分型——7种证型的食养思路

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-07 |
| **核心知识** | 高血压属于中医学"眩晕""头痛"范畴，主要病位在心肝脾肾，病性有实有虚，临床以**虚实夹杂**为主。共7种证型 |
| **适用人群** | 成人高血压人群（需辨证后选用相应食养方案） |
| **食养建议** | **7种证型及临床表现+食养方向**：（1）**肝火上炎证**——主症头晕胀痛、面红目赤、烦躁易怒；兼症耳鸣如潮、胁痛口苦、便秘溲黄；舌红苔黄脉弦数；→饮食以清淡为主，平肝潜阳。（2）**痰湿内阻证**——主症头重如裹；兼症胸脘痞闷、纳呆恶心、呕吐痰涎、身重困倦、少食多寐；苔腻脉滑；→清淡易消化、少食多餐，健脾运湿。（3）**瘀血内阻证**——主症头痛如刺、痛有定处；兼症胸闷心悸、手足麻木、夜间尤甚；舌质暗脉弦涩；→清淡温平，活血通络。（4）**阴虚阳亢证**——主症眩晕耳鸣、腰酸膝软、五心烦热；兼症头重脚轻、口燥咽干、两目干涩；舌红少苔脉细数；→清淡养阴生津，滋阴潜阳。（5）**肾精不足证**——主症心烦不寐、耳鸣腰酸；兼症心悸健忘、失眠梦遗、口干口渴；舌红脉细数；→偏温补，补益肝肾。（6）**气血两虚证**——主症眩晕时作、短气乏力、口干心烦；兼见面白自汗或盗汗、心悸失眠、纳呆腹胀便溏；舌淡脉细；→少食多餐、细软滋补，补益气血。（7）**冲任失调证**——妇女月经来潮或更年期前后头痛头晕；兼症心烦失眠、胁痛全身不适；血压波动；舌淡脉弦细；→清淡富含营养，调和冲任 |
| **食材/方剂** | 详见附录4各证型食养方举例（每证2方，共14个食养方） |
| **注意事项** | 辨证应由专业中医师进行；同一患者可能兼夹多种证型，需综合判断；证型可能随病程和治疗发生变化，应定期复评 |
| **关联KPK** | KPK-02（科学食养中的辨证施膳），KPK-06（疾病定义），KPK-11（食养方举例） |

## KPK-09：高血压水平分级与心血管风险分层

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-09 |
| **核心知识** | 高血压按血压水平分为3级（轻中重度），结合其他心血管危险因素进行风险分层（低危/中危/高危/很高危），指导管理强度 |
| **适用人群** | 成人高血压人群及基层卫生工作者 |
| **食养建议** | **血压水平分类**：（1）正常血压：SBP<120且DBP<80mmHg；（2）正常高值：120～139和/或80～89mmHg；（3）高血压：≥140和/或≥90mmHg；（4）1级（轻度）：140～159和/或90～99；（5）2级（中度）：160～179和/或100～109；（6）3级（重度）：≥180和/或≥110；（7）单纯收缩期高血压：≥140且<90。注：当SBP和DBP分属不同级别时以较高者为准。**心血管风险分层**（基于SBP/DBP水平+其他危险因素/疾病史）：无其他危险因素时：SBP130-139/DBP85-89→低危；SBP140-159/90-99→中危；SBP160-179/100-109→高危；SBP≥180/≥110→很高危。1~2个其他危险因素：各档提升一级。≥3个危险因素/靶器官损害/CKD3期/无并发症糖尿病：中高危起跳。临床并发症/CKD≥4期/有并发症糖尿病：均为高/很高危 |
| **食材/方剂** | 无特定食材，风险分层决定干预力度（生活方式vs药物治疗） |
| **注意事项** | CKD=慢性肾脏疾病；靶器官损害包括左心室肥厚、蛋白尿、视网膜病变等；家庭自测血压可作为诊室血压补充但诊断仍以诊室为准 |
| **关联KPK** | KPK-06（定义与流行病学），KPK-05（监测与自我管理） |
