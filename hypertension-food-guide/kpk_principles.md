# KPK 知识点：食养原则

## KPK-01：减钠增钾，饮食清淡

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-01 |
| **核心知识** | 钠盐摄入过多可增加高血压风险。增加膳食中钾摄入量可降低血压。所有高血压患者均应限制来源于各类食物的钠盐摄入 |
| **适用人群** | 成人高血压人群 |
| **食养建议** | 每人每日食盐摄入量逐步降至**5g以下**；增加富钾食物（新鲜蔬菜、水果、豆类）摄入。我国居民膳食中**75%以上的钠来自家庭烹调盐**，其次为高盐调味品；加工食品也是重要钠盐来源。清淡饮食，少吃含高脂肪、高胆固醇的食物。肾功能良好者可选择**高钾低钠盐**；不建议服用钾补充剂（包括药物）来降低血压；肾功能不全者补钾前应咨询医生。适当选择富含钙、镁的食物——钙摄入不足是我国居民普遍问题；镁对周围血管系统有血管扩张作用，可对抗高钠的升压作用。限制膳食脂肪和胆固醇（油炸食品、动物内脏），少吃加工红肉制品（培根、香肠、腊肠等） |
| **食材/方剂** | 高钾食物：新鲜蔬果、豆类、菌藻类、薯类；高钠需限制：咸菜、酱类、味精/鸡精/蚝油、加工肉制品、方便面/挂面 |
| **注意事项** | 减盐要循序渐进，逐步培养清淡口味；注意隐形钠（酱油含钠约5-7g/100ml，蚝油含钠约4-6g/100ml）；烹调时可用醋、柠檬汁、香料等替代部分盐 |
| **关联KPK** | KPK-02（合理膳食），KPK-08（食物选择） |

## KPK-02：合理膳食，科学食养

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-02 |
| **核心知识** | 平衡膳食应由五大类食物组成，高血压患者应在平衡膳食基础上调整优化食物种类和重量，遵循"辨证施膳""辨体施膳"原则 |
| **适用人群** | 成人高血压人群 |
| **食养建议** | **五大类食物**：①谷薯类（全谷物、薯类与杂豆）；②蔬菜水果；③动物性食物（畜禽鱼蛋奶）；④大豆类和坚果；⑤烹调油和盐。具体推荐量：（1）谷类200～300g/天（其中全谷物和杂豆**50～150g**），薯类50～100g/天；（2）深色蔬菜占总蔬菜量**一半以上**，蔬菜水果不能相互替代；（3）动物性食物120～200g/天，选鱼虾、禽、蛋、瘦肉，少选高盐高脂高胆固醇者；（4）奶类300～500g/天；（5）限制添加糖摄入；（6）减少食盐及含钠调味品（酱油、酱类、蚝油、鸡精、味精）。**7种证型辨证施膳**：肝火上炎→清淡为主平肝潜阳；痰湿内阻→清淡易消化少食多餐健脾运湿；瘀血内阻→清淡温平活血通络；阴虚阳亢→清淡养阴生津滋阴潜阳；肾精不足→偏温补补益肝肾；气血两虚→少食多餐细软滋补补益气血；冲任失调→清淡富含营养调和冲任 |
| **食材/方剂** | 全谷物（糙米、燕麦、荞麦、玉米、小米等）；新鲜蔬菜500g/天；新鲜水果200～350g/天；奶类（脱脂/低脂奶、酸奶）；大豆及制品（豆腐、豆浆等） |
| **注意事项** | 蔬菜水果不可相互替代（营养素构成不同）；少食用加入钠盐的谷类制品（咸味面包、方便面、挂面等）；推荐不同种类植物油交替使用 |
| **关联KPK** | KPK-01（减钠增钾），KPK-03（吃动平衡），KPK-07（中医分型） |

## KPK-03：吃动平衡，健康体重

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-03 |
| **核心知识** | 推荐将体重维持在健康范围内；超重和肥胖是高血压患病的重要危险因素，尤其是中心性肥胖 |
| **适用人群** | 成人高血压人群（尤其超重/肥胖者） |
| **食养建议** | **健康体重标准**：BMI **18.5～23.9kg/m²**（65岁以上老年人可适当增加）；男性腰围**＜85cm**，女性腰围**＜80cm**。超重和肥胖者应减重：控制能量摄入（比原来减少**300～500kcal/天**），同时控制高能量食物（高脂肪食物、含糖饮料、酒类等）。我国18～49岁成年人轻身体活动水平每日平均能量需要量：男性**2250kcal**，女性**1800kcal**。运动方面：每周累计**2.5～5小时中等强度有氧活动**或1.25～2.5小时高强度有氧活动。非高血压人群或高血压患者除日常活动外，应有每周**4～7天**、每天累计**30～60分钟**的中等强度身体活动。运动可以改善血压水平 |
| **食材/方剂** | 控制能量密度：减少油炸食品、甜点、含糖饮料；增加膳食纤维丰富的食物（全谷物、蔬菜）以增强饱腹感 |
| **注意事项** | 减重速度不宜过快，建议每周减轻0.5～1kg；避免极端节食；BMI<18.5者不适用减重建议 |
| **关联KPK** | KPK-02（合理膳食），KPK-05（监测血压自我管理） |

## KPK-04：戒烟限酒，心理平衡

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-04 |
| **核心知识** | 戒烟可降低心血管疾病风险。过量饮酒显著增加高血压发病风险，且随饮酒量增加而增加。精神紧张可激活交感神经使血压升高 |
| **适用人群** | 成人高血压人群 |
| **食养建议** | **戒烟**：彻底戒烟，避免被动吸烟。**限酒**：即使少量饮酒也会造成不良影响；建议高血压患者**不饮酒**，饮酒者尽量戒酒。**心理平衡**：减轻精神压力，进行压力管理（认知行为干预）；规律作息，保证充足睡眠，不熬夜；必要时到专业医疗机构就诊，避免因精神压力导致血压波动 |
| **食材/方剂** | 无特定食材，涉及生活方式干预 |
| **注意事项** | 突然戒酒可能出现戒断反应，建议在医师指导下逐步减量；失眠者可尝试睡前放松训练或饮用无咖啡因热饮 |
| **关联KPK** | KPK-05（监测血压） |

## KPK-05：监测血压，自我管理

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-05 |
| **核心知识** | 定期监测血压，了解血压数值及达标状态，遵医嘱进行生活方式干预和长期治疗。可根据自身健康状况选择适宜的膳食模式 |
| **适用人群** | 成人高血压人群 |
| **食养建议** | 根据心血管总体风险及血压水平（见附录5分级标准）进行随诊。**三种推荐膳食模式**：（1）**得舒饮食（DASH）**——富含新鲜蔬菜水果、低脂/脱脂乳制品、禽肉鱼大豆坚果和全谷物，限制含糖饮料和红肉，饱和脂肪酸和胆固醇水平低，富钾镁钙矿物质和优质蛋白膳食纤维；（2）**东方健康膳食模式**——东南沿海地区代表性膳食模式，特点为清淡少盐、食物多样、谷物为主，蔬菜水果充足，鱼虾水产丰富，奶类豆类丰富，身体活动量较高；（3）**中国心脏健康膳食（CHH-Diet）**——相比普通城市膳食减少钠和脂肪摄入，增加蛋白质、碳水化合物、钾、镁、钙和膳食纤维摄入 |
| **食材/方剂** | DASH饮食典型食材：燕麦、菠菜、香蕉、脱脂牛奶、三文鱼、杏仁、豆类；CHH-Diet：全谷物、瘦禽肉、鱼类、大豆、大量蔬菜水果 |
| **注意事项** | 家庭自测血压应固定时间（早晚各1次）、固定体位（坐位休息5分钟后测量）；记录血压日记有助于医师调整治疗方案 |
| **关联KPK** | KPK-03（吃动平衡），KPK-06（疾病定义），KPK-09（血压分级） |
