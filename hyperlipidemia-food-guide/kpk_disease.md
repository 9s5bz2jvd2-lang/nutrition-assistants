# KPK 知识点：疾病背景与附录5

## KPK-14：高脂血症定义、分类与流行病学

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-14 |
| **核心知识** | 近30年来我国高脂血症患病率明显增加，18岁及以上居民患病率高达35.6%，是高脂血症是高血压、糖尿病、冠心病、脑卒中的重要危险因素 |
| **适用人群** | 全人群（科普背景知识） |
| **食养建议** | 血脂是血清胆固醇、甘油三酯和类脂（如磷脂）等的总称。高脂血症（血脂异常）通常指血清中总胆固醇和（或）甘油三酯水平升高，也包括低高密度脂蛋白胆固醇血症。根据空腹静脉血清检测指标分为4种：高胆固醇血症（TC≥5.2mmol/L）、高甘油三酯血症（TG≥1.7mmol/L）、高低密度脂蛋白胆固醇血症（LDL-C≥3.4mmol/L）、低高密度脂蛋白胆固醇血症（HDL-C＜1.0mmol/L）。 |
| **食材/方剂** | 无 |
| **注意事项** | 当上述血脂指标一项及以上异常即可诊断为血脂异常。高脂血症危险因素多与不合理膳食相关（如过量饱和脂肪酸或反式脂肪酸摄入）。膳食营养通过调节血脂、血压或血糖水平影响动脉粥样硬化的发生。**ICD-11疾病编码为5C80等**。病因分为两类：①继发性高脂血症——由其他疾病引起；②原发性高脂血症——由不良生活方式（高能量/高脂饮食/过度饮酒/缺乏运动）或基因突变造成。参考指南：《中国成人血脂异常防治指南》（2016年修订版）、《血脂异常中西医结合诊疗专家共识》（2017年）、《欧洲血脂异常管理指南》（2016年版）。 |
| **关联KPK** | KPK-15（中医认识与分型），KPK-16（附录5血脂分层标准） |

## KPK-15：中医对高脂血症的认识及分型

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-15 |
| **核心知识** | 中医将高脂血症纳入"血瘀""痰湿""脂膏"范畴，病因在于饮食不节、嗜食油腻甘甜、醇酒厚味、情志失调、过逸少劳，属本虚标实之证 |
| **适用人群** | 成人高脂血症人群（中医辨证参考） |
| **食养建议** | 6种常见辨证分型及临床表现：1.痰浊内阻型：身体肥胖，肢体沉重感，头昏多眠，容易困倦，胸闷气短，大便黏或不成形，舌体胖大，舌苔粘腻，脉滑。2.痰瘀互结型：身体肥胖，肢体沉重感，头昏多眠，容易困倦，胸刺痛或闷痛，口唇暗紫，大便黏腻，舌体胖大，舌苔粘腻，或舌质紫暗，或舌体有瘀点瘀斑，脉滑或涩。3.气滞血瘀型：胸部或胁部胀满，或针刺样疼痛，情绪低落或急躁易怒，喜欢长叹气，口唇紫暗，舌暗红，有瘀点或瘀斑，脉细涩。4.气虚血瘀型：气短乏力，精神疲倦，少言懒言，胸部或胁部针刺样疼痛，活动后诱发或加重，出汗多，舌淡暗或淡紫或有瘀斑、瘀点，脉涩。5.肝肾阴虚型：头晕耳鸣，腰酸腿软，手心、脚心发热，心烦失眠，健忘多梦，舌红，舌苔少，脉细数。6.脾虚湿盛型：身体困倦，大便不成形或腹泻，饮食无味，食后腹胀，舌淡，舌体胖大有齿痕，舌苔色白粘腻，脉细弱或濡缓。 |
| **食材/方剂** | 无（食药物质选择见附录1表1.2） |
| **注意事项** | 该病属于本虚标实之证，以痰瘀为标，正虚为本。不同证型需采用不同的食养方案和食药物质。 |
| **关联KPK** | KPK-05（辨证施膳），KPK-10（证型食药物质），KPK-12（证型食谱），KPK-13（证型食养方） |

## KPK-16：附录5 — 血脂水平分层和血脂异常的临床分类

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-16 |
| **核心知识** | ASCVD一级预防人群的血脂水平分层标准和血脂异常的4种临床分类 |
| **适用人群** | 成人高脂血症人群及医疗专业人员 |
| **食养建议** | **ASCVD一级预防血脂分层标准（mmol/L）**：总胆固醇（TC）合适水平＜5.2，边缘升高≥5.2且＜6.2，升高≥6.2。甘油三酯（TG）合适水平＜1.7，边缘升高≥1.7且＜2.3，升高≥2.3。低密度脂蛋白胆固醇（LDL-C）理想水平＜2.6，合适水平＜3.4，边缘升高≥3.4且＜4.1，升高≥4.1。高密度脂蛋白胆固醇（HDL-C）降低＜1.0。非高密度脂蛋白胆固醇（非HDL-C）理想水平＜3.4，合适水平＜4.1，边缘升高≥4.1且＜4.9，升高≥4.9。 |
| **食材/方剂** | 无 |
| **注意事项** | 血脂异常的4种临床分类：高胆固醇血症（TC增高）、高甘油三酯血症（TG增高）、混合型高脂血症（TC+TG均增高）、低高密度脂蛋白胆固醇血症（HDL-C降低）。 |
| **关联KPK** | KPK-14（定义与分类） |
