# KPK 知识点：附录知识

## KPK-09：食物选择（附录1）

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-09 |
| **核心知识** | 高脂血症人群应遵循推荐食物名单，选择低脂、高膳食纤维食物，减少高脂肪、高盐、高糖食物 |
| **适用人群** | 成人高脂血症人群 |
| **食养建议** | 谷薯类宜选糙米、全麦面粉、玉米、荞麦、燕麦、藜麦等全谷物；肉类宜选鱼虾类、瘦肉、去皮禽肉，限制肥肉、加工肉制品、鱼籽、蟹黄、动物内脏等；蛋类可选鸡蛋、鸭蛋等，限制咸蛋；奶类选脱脂奶、低脂奶、鲜牛奶、低糖酸奶等，限制奶油、黄油；大豆及制品选黄豆、黑豆、豆腐、豆腐干等，限制油豆腐皮、豆腐泡等油炸豆制品；蔬菜选新鲜蔬菜，限制腌制蔬菜；食用油选紫苏油、亚麻籽油、橄榄油、茶籽油、菜籽油等，限制棕榈油、椰子油、猪油、牛油、羊油等动物油；调味品用低钠盐（每日≤5g），限制酱类、腐乳等高盐调味品。 |
| **食材/方剂** | 详见食物选择表1.1 |
| **注意事项** | 各食物类别均有宜选择和限制品种的具体清单，需一一对照执行。 |
| **关联KPK** | KPK-02（调控脂肪），KPK-03（食物多样），KPK-04（少盐控糖），KPK-08（会看慧选） |

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-10 |
| **核心知识** | 不同中医证型的高脂血症人群应选用不同的食药物质，以辅助降低血脂 |
| **适用人群** | 成人高脂血症人群（按中医证型选择） |

| **食材/方剂** | 每种证型对应不同食药物质清单。†试点物质（非试点地区限执业医师使用），#新食品原料。 |
| **注意事项** | 1.在限定使用范围和剂量内作为食药物质。2.食用方法请咨询医生、营养指导人员等专业人员。3.试点物质以†标记，非试点地区限执业医师使用。4.新食品原料以#标记，每种每天食用量不能超过相关规定中使用量，多种联合食用时宜酌情控制。 |
| **关联KPK** | KPK-05（辨证施膳），KPK-11（附录2食物交换表） |

## KPK-11：常见食物交换表（附录2）

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-11 |
| **核心知识** | 8类食物的等量交换表（每份90kcal），使用者可参考交换表和食谱示例相互交换、合理搭配 |
| **适用人群** | 成人高脂血症人群及营养指导人员 |
| **食养建议** | 8类食物交换表包括：谷薯类、蔬菜类、水果类、肉类、坚果类、大豆乳制品类、调味料盐含量类、油脂类。每份均以90kcal为标准能量单位。谷薯类每份23～27g（谷物干重）；蔬菜类每份240～260g（综合）；水果类每份140～160g（水果综合）；肉类每份40～60g（畜肉综合）；坚果类每份12～16g（高脂类）；大豆类每份18～22g；调味料按2000mg钠（5g盐）等量交换；油脂类每份9～11g（1汤勺）。 |
| **食材/方剂** | 8个交换表的详细食物清单 |
| **注意事项** | 如混食多种蔬菜/水果，选择综合份量；如单选某类，按类确定份量。不便判断脂肪含量时选择畜肉（综合）的份量。五花肉、肥肉宜减少食用频次或摄入总量。 |
| **关联KPK** | KPK-03（食物多样），KPK-09（食物选择），KPK-12（食谱示例） |

## KPK-12：不同证型食谱示例（附录3）

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-12 |
| **核心知识** | 针对6种中医证型分别给出食谱示例（每证型3个示例，共18套），适合轻体力活动成人高脂血症人群，每日能量1700～1900kcal |
| **适用人群** | 成人高脂血症人群（按中医证型选择） |
| **食养建议** | 食谱坚持食物多样、能量适宜、控制脂肪、主食定量、蔬果丰富、肉类适当的原则。能量范围1700～1900kcal。6证型为：痰浊内阻型（3例）、痰瘀互结型（3例）、气滞血瘀型（3例）、气虚血瘀型（3例）、肝肾阴虚型（3例）、脾虚湿盛型（3例）。每套食谱含早餐、茶饮、中餐、加餐、晚餐及全天油盐用量。全天植物油20g，盐5g。宏量营养素比例：蛋白质15%～20%，碳水化合物50%～60%，脂肪20%～25%。 |
| **食材/方剂** | 详见完整食谱数据文件（recipes_data.md） |
| **注意事项** | 食谱以生重计。使用者可结合自身活动量调整。可参考食物交换表进行互换。*食物物质、#新食品原料需注意用法用量。 |
| **关联KPK** | KPK-10（证型食药物质），KPK-11（食物交换表），KPK-13（食养方举例） |

## KPK-13：不同证型食养方举例（附录4）

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-13 |
| **核心知识** | 6种证型各有对应的食养茶饮和食养方，共39个（18茶饮+21膳方），均需辨证选用 |
| **适用人群** | 成人高脂血症人群（按中医证型选择） |
| **食养建议** | 痰浊内阻型：3茶饮（山楂菊花决明子茶、三鲜茶、三鲜饮）+ 3食养方（桔红蜇皮鸭肉汤、海带冬瓜薏苡仁汤、冬瓜莲蓬薏苡仁煲瘦肉）。痰瘀互结型：3茶饮（山楂薏苡仁饮、山楂菊花决明子茶、海带绿豆水）+ 4食养方（橘皮佛手山楂粥、山楂西兰花炒肉片、鲫鱼山楂萝卜汤、山楂黑木耳乌鸡汤）。气滞血瘀型：3茶饮（山楂橘皮饮、菊楂决明饮、山楂玫瑰花茶）+ 2食养方（猪肉炒山楂、佛手桃仁煲瘦肉）。气虚血瘀型：3茶饮（山楂甘草茶、山楂甘草薏苡仁饮、荷叶山楂饮）+ 3食养方（芪参鲤鱼汤、桃仁鸡、归芪鸡汤）。肝肾阴虚型：3茶饮（杞菊饮、山楂菊花决明子茶、荷叶夏枯草枸杞茶）+ 4食养方（枸杞芝麻蔬菜饼、黄精枸杞焖鸭、枸杞芝麻粥、枸杞子炖兔肉）。脾虚湿盛型：3茶饮（健脾饮、三花橘皮茶、山楂橘皮茶）+ 5食养方（山药芡薏粥、山药茯苓煲乳鸽、扁豆大枣蒸海鱼、茯苓赤小豆粥、芡实八珍糕）。 |
| **食材/方剂** | 详见完整食养方数据（附录4） |
| **注意事项** | 各种茶饮和食养方均有特定禁忌（孕妇慎用/禁用、过敏者禁用等）。部分药材有使用限制（如†限执业医师使用、#新食品原料限量）。食养方用法用量：佐餐食用，1人1次量，可食用7～10天。 |
| **关联KPK** | KPK-10（证型食药物质），KPK-12（食谱示例） |
