## 指南出处

- **全称**：《成人高脂血症食养指南（2023年版）》
- **发布单位**：国家卫生健康委办公厅
- **制定依据**：《健康中国行动（2019—2030年）》《国民营养计划（2017—2030年）》

## 功能特点

- **风险评估**：血脂异常自评、高危因素识别与分层管理
- **食养原则**：8条官方原则的详细解读与实操建议
- **脂质调控**：总胆固醇/甘油三酯/LDL-C/HDL-C的饮食干预策略
- **中医辨证**：6种证型（痰浊内阻/痰瘀互结/气滞血瘀/气虚血瘀/肝肾阴虚/脾虚湿盛）×3示例 = **18套食谱**
- **食养方库**：39个食养方（18茶饮+21膳方），按证型分类选用
- **地区适配**：7大地区×4季节×3能量等级 = **84个日食谱**完整收录
- **食物选择**：推荐/限制/避免食物清单 + 替代方案
- **科普风格**：说人话、给具象量、指误区——精准但不端着

### 关键数据速查表

> 📌 以下为**一般人群**推荐量；高胆固醇血症者见括号内强化值。

| 项目 | 推荐量 | 通俗理解 |
|------|--------|---------|
| 烹调油 | ≤25g/天（高脂血症者≤20g） | 两汤匙封顶 |
| 食盐 | ≤5g/天 | 一个啤酒瓶盖 |
| 膳食胆固醇 | ＜300mg/天（高胆固醇血症＜200mg） | 1个大鸡蛋≈200~250mg，高脂血症者每天不超过1个蛋 |
| 总膳食纤维 | 25～40g/天 | 一斤菜+半斤水果+杂粮 |
| 其中可溶性纤维 | 10～25g/天 | 燕麦+豆类+蔬果搭配 |
| 新鲜蔬菜 | 500g/天 | 深色蔬菜占一半以上 |
| 新鲜水果 | 200～350g/天 | 一个苹果+一小碗浆果 |
| 全谷物/杂豆 | 50～150g/天 | 一碗糙米饭或杂粮粥 |
| 饱和脂肪供能比 | ＜总能量10%（高胆固醇血症＜7%） | 别吃猪油牛油 |
| 反式脂肪酸 | ≤2g/天（＜总能量1%） | 少买含氢化植物油的加工食品 |
| 每周深海鱼 | ≥2次 | 三文鱼/带鱼/鲭鱼轮着吃 |
| TC合适水平 | ＜5.2mmol/L | 总胆固醇不高过5.2 |
| TG合适水平 | ＜1.7mmol/L | 甘油三酯不高过1.7 |
| LDL-C合适水平 | ＜3.4mmol/L（理想＜2.6） | 坏胆固醇不高过3.4 |
| HDL-C | ≥1.0mmol/L | 好胆固醇不低于1.0 |
