# KPK 知识点：食养原则

## KPK-01：吃动平衡，保持健康体重

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-01 |
| **核心知识** | 通过改善膳食结构、控制能量摄入、增加运动，维持健康体重，减少体脂含量，有利于血脂控制 |
| **适用人群** | 成人高脂血症人群（含超重和肥胖人群） |
| **食养建议** | 超重和肥胖人群：每天减少300～500kcal能量摄入，使体重减少10%以上。每周5～7次体育锻炼或身体活动，每次30分钟中等及以上强度运动（快走、跑步、游泳、爬山、球类等），每天锻炼至少消耗200kcal。稳定性动脉粥样硬化性心血管疾病患者应先进行运动负荷试验。 |
| **食材/方剂** | 无 |
| **注意事项** | 运动强度宜循序渐进、量力而行，以运动后第2天感觉精力充沛、无不适感为宜。部分不宜进行运动人群除外。 |
| **关联KPK** | KPK-02（调控脂肪），KPK-04（少盐控糖） |

## KPK-02：调控脂肪，少油烹饪

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-02 |
| **核心知识** | 限制总脂肪、饱和脂肪、胆固醇和反式脂肪酸的摄入，是防治高脂血症和动脉粥样硬化性心血管病的重要措施 |
| **适用人群** | 成人高脂血症人群 |
| **食养建议** | 脂肪摄入量占总能量20%～25%（高甘油三酯血症者应尽可能减少）。以每日能量1800～2000kcal为例，全天脂肪摄入量40～55g（含烹调油、动物性食品及坚果等），每日烹调油≤25g。饱和脂肪占总能量＜10%（高胆固醇血症者＜7%）。胆固醇每日＜300mg（高胆固醇血症者＜200mg）。反式脂肪酸占总能量＜1%（每天≤2g）。适当增加不饱和脂肪酸，特别是n-3系列多不饱和脂肪酸。 |
| **食材/方剂** | 紫苏油、亚麻籽油、核桃油、橄榄油、茶籽油、菜籽油、葵花籽油、玉米油、芝麻油、豆油、花生油等 |
| **注意事项** | 减少油炸、油煎等多油烹饪方法，多选择**蒸、煮、汆、拌**等方式。少吃富含胆固醇的食物，如动物脑和动物内脏。减少或避免部分氢化植物油等含反式脂肪酸的食物。 |
| **关联KPK** | KPK-01（吃动平衡），KPK-09（食物选择） |

## KPK-03：食物多样，蛋白质和膳食纤维摄入充足

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-03 |
| **核心知识** | 在控制总能量及脂肪的基础上，选择食物多样的平衡膳食模式，保证蛋白质和膳食纤维摄入充足 |
| **适用人群** | 成人高脂血症人群 |
| **食养建议** | 食物每天≥12种，每周≥25种。碳水化合物占总能量50%～60%（以1800～2000kcal为例，约225～300g）。膳食纤维每日25～40g（其中水溶性膳食纤维7～13g）。新鲜蔬菜每日500g（深色蔬菜占一半以上）。新鲜水果每日200～350g。蛋白质摄入充足：选择脂肪较低的鱼虾类、去皮禽肉、瘦肉等；奶类选脱脂或低脂牛奶。每天摄入含25g大豆蛋白的食品。 |
| **食材/方剂** | 全谷物、杂豆类、蔬菜（深色蔬菜占一半以上）、水果、鱼虾类、去皮禽肉、瘦肉、脱脂/低脂牛奶、豆制品 |
| **注意事项** | 适当控制精白米面摄入。高膳食纤维可降低血胆固醇水平，提高胰岛素敏感性。 |
| **关联KPK** | KPK-02（调控脂肪），KPK-10（常见食物交换表） |

## KPK-04：少盐控糖，戒烟限酒

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-04 |
| **核心知识** | 为预防高脂血症相关并发症（高血压、糖尿病、冠心病、脑卒中），需将血脂、血压、血糖控制在理想水平，控制盐和糖的摄入 |
| **适用人群** | 成人高脂血症人群 |
| **食养建议** | 食盐每日≤5g。少吃酱油、鸡精、味精、咸菜、咸肉、酱菜等高盐食品。添加糖摄入不应超过总能量的10%（肥胖和高甘油三酯血症者应更低）。生活作息规律，戒烟限酒（即使少量饮酒也可使高甘油三酯血症者甘油三酯水平进一步升高）。 |
| **食材/方剂** | 无 |
| **注意事项** | 限制单糖和双糖摄入，少吃甜食。完全戒烟和避免二手烟，有利于预防动脉粥样硬化性心血管疾病。 |
| **关联KPK** | KPK-01（吃动平衡），KPK-02（调控脂肪） |

## KPK-05：因人制宜，辨证施膳

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-05 |

| **适用人群** | 成人高脂血症人群（各中医证型） |

## KPK-06：因时制宜，分季调理

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-06 |
| **核心知识** | 在四时节律影响下，人体血脂水平存在差异，不同季节的食养有不同要求 |
| **适用人群** | 成人高脂血症人群 |
| **食养建议** | 春季：护阳保肝，多食时令蔬菜（芹菜、芦笋等），适当食用佛手、生麦芽、菊花等，忌过食寒凉黏滞肥腻。初夏：益气清心，适当食用鸭肉、鱼类、兔肉、小麦、绿豆、豆腐等。长夏：清利湿热、健运脾胃，食用橘皮、薏苡仁、白扁豆、赤小豆、莱菔子等。秋季：滋阴润肺，食用桑椹、黑芝麻、乌梅、百合等，少吃辛辣煎炸油腻。冬季：散寒邪、补肾阳，适当食用羊肉等温性食物及枸杞子、黄精、山茱萸等，忌食生冷之物。 |
| **食材/方剂** | 春季：佛手、生麦芽、菊花；长夏：橘皮、薏苡仁、白扁豆、赤小豆、莱菔子；秋季：桑椹、黑芝麻、乌梅、百合；冬季：枸杞子、黄精、山茱萸、羊肉 |
| **注意事项** | 人与自然是一个有机整体，膳食要顺应四时节律。 |
| **关联KPK** | KPK-05（辨证施膳），KPK-07（因地制宜） |

## KPK-07：因地制宜，合理搭配

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-07 |
| **核心知识** | 根据地域调整膳食，不同地区高脂血症人群的中医体质和膳食习惯不同 |
| **适用人群** | 成人高脂血症人群（按地区） |
| **食养建议** | 北方地区（温带季风气候）：多食新鲜蔬果、鱼虾类、奶类、豆类，减少腌制蔬菜；适当食用橘皮、薏苡仁、白扁豆、赤小豆、莱菔子、山楂、桃仁、沙棘等。南方地区（亚热带季风气候）：控制油盐，增加粗粮（紫薯、玉米、黑米、大麦、青稞等）；适当食用人参、白扁豆、薏苡仁、山药、大枣、麦芽、茯苓等。西北地区（温带大陆性气候）：减少牛羊肉，多食蔬菜水果；适当食用枸杞子、桑椹、菊花、黑芝麻、百合、乌梅、决明子等。青藏地区（高原山地气候）：多食去皮禽肉、鱼等动物蛋白，补充优质植物蛋白如大豆蛋白，增加蔬菜水果。 |
| **食材/方剂** | 各地不同：北方→橘皮/薏苡仁/山楂/桃仁等；南方→人参/白扁豆/山药/茯苓等；西北→枸杞子/桑椹/决明子/黑芝麻等；青藏→大豆蛋白 |
| **注意事项** | 不同地区气候、环境影响居民膳食习惯和生理特征差异。 |
| **关联KPK** | KPK-06（因时制宜），KPK-09（食物选择） |

## KPK-08：会看慧选，科学食养，适量食用食药物质

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-08 |
| **核心知识** | 通过看食品营养标签选择适合的食品，适量食用富含植物化学物及有辅助降脂作用的食药物质 |
| **适用人群** | 成人高脂血症人群 |
| **食养建议** | 通过看营养标签选择脂肪含量低的食品，了解能量和营养成分（碳水化合物、蛋白质、膳食纤维、钠等）。每日可摄入2g左右植物甾醇。不同证型高脂血症人群可按附录推荐适量食用食药物质。 |
| **食材/方剂** | 富含植物甾醇/多糖的食物：大豆、洋葱、香菇以及深色蔬果等。不同证型食药物质详见附录1表1.2。 |
| **注意事项** | 食药物质及新食品原料食用量应符合相关要求。已知过敏者、正在服用药物有禁忌者，应在医师、执业药师及营养指导人员指导下使用。新食品原料以#标记，每种每天食用量不能超过规定。 |

## KPK-09：食物选择（附录1）

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-09 |
| **核心知识** | 高脂血症人群应遵循推荐食物名单，选择低脂、高膳食纤维食物，减少高脂肪、高盐、高糖食物 |
| **适用人群** | 成人高脂血症人群 |
| **食养建议** | 谷薯类宜选糙米、全麦面粉、玉米、荞麦、燕麦、藜麦等全谷物；肉类宜选鱼虾类、瘦肉、去皮禽肉，限制肥肉、加工肉制品、鱼籽、蟹黄、动物内脏等；蛋类可选鸡蛋、鸭蛋等，限制咸蛋；奶类选脱脂奶、低脂奶、鲜牛奶、低糖酸奶等，限制奶油、黄油；大豆及制品选黄豆、黑豆、豆腐、豆腐干等，限制油豆腐皮、豆腐泡等油炸豆制品；蔬菜选新鲜蔬菜，限制腌制蔬菜；食用油选紫苏油、亚麻籽油、橄榄油、茶籽油、菜籽油等，限制棕榈油、椰子油、猪油、牛油、羊油等动物油；调味品用低钠盐（每日≤5g），限制酱类、腐乳等高盐调味品。 |

# KPK 知识点：疾病背景与附录5

## KPK-14：高脂血症定义、分类与流行病学

| 字段 | 内容 |
|------|------|
| **KPK ID** | KPK-14 |
| **核心知识** | 近30年来我国高脂血症患病率明显增加，18岁及以上居民患病率高达35.6%，是高脂血症是高血压、糖尿病、冠心病、脑卒中的重要危险因素 |
| **适用人群** | 全人群（科普背景知识） |
| **食养建议** | 血脂是血清胆固醇、甘油三酯和类脂（如磷脂）等的总称。高脂血症（血脂异常）通常指血清中总胆固醇和（或）甘油三酯水平升高，也包括低高密度脂蛋白胆固醇血症。根据空腹静脉血清检测指标分为4种：高胆固醇血症（TC≥5.2mmol/L）、高甘油三酯血症（TG≥1.7mmol/L）、高低密度脂蛋白胆固醇血症（LDL-C≥3.4mmol/L）、低高密度脂蛋白胆固醇血症（HDL-C＜1.0mmol/L）。 |
| **食材/方剂** | 无 |
| **注意事项** | 当上述血脂指标一项及以上异常即可诊断为血脂异常。高脂血症危险因素多与不合理膳食相关（如过量饱和脂肪酸或反式脂肪酸摄入）。膳食营养通过调节血脂、血压或血糖水平影响动脉粥样硬化的发生。**ICD-11疾病编码为5C80等**。病因分为两类：①继发性高脂血症——由其他疾病引起；②原发性高脂血症——由不良生活方式（高能量/高脂饮食/过度饮酒/缺乏运动）或基因突变造成。参考指南：《中国成人血脂异常防治指南》（2016年修订版）、《血脂异常中西医结合诊疗专家共识》（2017年）、《欧洲血脂异常管理指南》（2016年版）。 |
| **关联KPK** | KPK-15（中医认识与分型），KPK-16（附录5血脂分层标准） |
