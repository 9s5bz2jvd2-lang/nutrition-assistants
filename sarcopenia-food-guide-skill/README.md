## 指南出处

- **全称**：《成人肌少症食养指南（2026年版）》
- **发布单位**：国家卫生健康委办公厅
- **制定依据**：《健康中国行动（2019—2030年）》《国民营养计划（2017—2030年）》
- **参考共识**：欧洲肌少症共识（EWGSOP 2014/2019）、亚洲肌少症工作组共识（AWGS 2019/2025）、中国肌少症系列专家共识（2015-2025）等

## 功能特点

- **风险评估**：SARC-F量表5步自评，握力/小腿围/步速切点查询
- **食养原则**：7条官方原则的详细解读与实操建议
- **蛋白质指导**：精确摄入量计算、三餐分配方案、食物等价交换
- **营养补充**：乳清蛋白/亮氨酸/CaHMB/维生素D的剂量与注意事项
- **地区食谱**：7大地区×4季节×3能量等级 = **84个日食谱**完整收录
- **中医食养**：3种证型12个食养方的配方与辨证选用
- **合并疾病**：肥胖/糖尿病/肾病/肿瘤的个体化食养调整
- **科普风格**：说人话、给具象量、指误区——精准但不端着

## 快速开始

### 5步自评（SARC-F量表）

1. 举起9斤（4.5kg）的东西有无困难？（0/1/2分）
2. 自己走过一个房间有无困难？（0/1/2分）
3. 从椅子上站起来费劲不？（0/1/2分）
4. 爬一层楼（约10个台阶）有无困难？（0/1/2分）
5. 过去1年摔过几次跤？（0次=0，1-3次=1，≥4次=2）

**总分≥4分** → 肌肉可能在"缩水"，建议去做个正规检查

### 关键数据速查

| 项目 | 推荐量 | 通俗理解 |
|------|--------|---------|
| 蛋白质（肌少症） | 1.2-1.5g/kg/天 | 60kg的人每天72-90g |
| 每餐蛋白质 | 20-25g | 约1个蛋+1杯奶的量 |
| 优质蛋白占比 | ≥60% | 三份蛋白两份得是"好的" |
| 乳清蛋白补充 | 20-40g/天 | 半到一勺蛋白粉 |
| 小腿围（男/女） | ≥34cm/≥33cm | 比矿泉水瓶粗不了多少就要当心 |
| 握力（男≥65岁） | ≥28.0kg | 拧不开瓶盖要注意 |
| 握力（女≥65岁） | ≥18.0kg | 提不了5斤重物要查一查 |
