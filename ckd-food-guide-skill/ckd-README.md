# 营养学 | 慢性肾脏病食养助手 (CKD Food Guide)

> 基于《成人慢性肾脏病食养指南（2024年版）》| v1.0.0

## ✨ 这是什么？

这是一个 **AI 科普对话 Skill**，将国家卫生健康委发布的《成人慢性肾脏病食养指南（2024年版）》完整蒸馏为结构化的知识库。安装后，AI 助手就能像社区营养师一样，为慢性肾脏病患者提供科学的膳食指导和中医食养建议。

## 🎯 核心功能

### 1️⃣ 分期差异化饮食指导
- **CKD 1-2期**：植物性食物为主，正常蛋白，强调预防
- **CKD 3-5期（非透析）**：低蛋白饮食（0.6g/kg/d），主食优化，减轻肾脏负担
- **CKD 5期透析阶段**：高蛋白饮食（1.0-1.2g/kg/d），仍控钾磷钠

### 2️⃣ 中医辨证施膳
覆盖 **7 种证型**的食养方推荐：

| 类型 | 证型 | 推荐方向 |
|------|------|---------|
| 本虚 | 气虚证 | 山药、茯苓、黄精 |
| 本虚 | 血虚证 | 阿胶 |
| 本虚 | 阴虚证 | 雪梨、山药、蜂蜜、银耳 |
| 本虚 | 阳虚证 | 当归（香辛料）、阿胶 |
| 标实 | 湿证 | 赤小豆、金银花、菊花 |
| 标实 | 热证 | 同上 + 季节调整 |
| 标实 | 瘀证 | 桃仁、山楂 |

### 3️⃣ 84 套全天食谱
**7 地区 × 3 分期 × 4 季节 = 84 套**

| 地区 | 特色 |
|------|------|
| 东北 | 适应寒冷气候，高能量密度食材 |
| 西北 | 温补滋润，干旱气候适配 |
| 华北 | 四季分明，面食文化 |
| 华东 | 鱼米之乡，清淡鲜美 |
| 华中 | 微辣口味，藕粉/热干面等特色 |
| 西南 | 麻辣风味，酸辣粉/肥肠粉等 |
| 华南 | 清热利湿，岭南特色 |

每套食谱含完整营养数据：**能量 / 蛋白质 / 脂肪 / 碳水化合物 / 钠 / 钾 / 磷**

### 4️⃣ 数据速查工具
- **表1.1**：常见食物钠/钾/磷/蛋白质含量及磷/蛋白质比值（按比值分组）
- **食物交换表**：谷薯类/蔬菜类/水果类/肉蛋奶类/油脂坚果类

### 5️⃣ 7 种症状食养方

| 症状 | 证型 | 食养方 |
|------|------|--------|
| 水肿 | 气虚湿盛 | 黄芪鲤鱼汤 |
| 尿频尿急 | 湿热下注 | 薏米绿豆汤 |
| 贫血 | 气血亏虚 | 桂圆红枣粥 |
| 腹泻纳差 | 脾虚湿盛 | 扁豆山药粥 |
| 便秘 | 阴虚/实热 | 三仁润肠饮 / 槐花茶 |
| 咽干咽痛 | 阴虚火旺 | 清热利咽茶 |
| 腰膝酸软夜尿多 | 肾阳亏虚 | 猪腰二仁汤 |

## 📦 安装方法

### 方法一：一键安装脚本

**Windows:**
```bash
install.bat
```

**Linux/macOS:**
```bash
bash install.sh
```

### 方法二：手动安装
1. 将本仓库所有文件复制到你的 Skills 目录：
   - WorkBuddy: `~/.workbuddy/skills/custom/ckd-food-guide-skill/`
   - 其他：参照你的客户端说明
2. 重启 AI 助手或重新加载 Skills

## 💡 使用示例

### 问："我刚查出慢性肾脏病3期，饮食要注意什么？"
→ AI 会给出 CKD 3-5 期的核心饮食要点：低蛋白饮食（0.6g/kg/d）、主食优化、控盐控磷控钾、具体食物推荐和禁忌。

### 问："透析患者能吃水果吗？"
→ AI 会根据透析期的特点，给出适合的水果推荐和需谨慎选择的高钾水果列表。

### 问："我爸爸有肾病水肿，中医有什么食疗方？"
→ AI 会先辨证，推荐对应证型的食养方（如气虚湿盛→黄芪鲤鱼汤），附配料和制法。

## 🚨 免责声明

1. 本 Skill 内容来源为国家卫生健康委发布的《成人慢性肾脏病食养指南（2024年版）》，**不可替代医疗诊断和药物治疗**。
2. 适用人群为成人慢性肾脏病患者（CKD 1-5期），不适用于儿童、妊娠期女性及急性肾损伤患者。
3. CKD 3期以上患者使用食养方前，应结合血钾、血磷、尿酸等指标，在医生指导下合理选用。
4. 带 `*` 号标记的食材为**食药物质**，已知过敏者禁用；正在服用药物者应注意可能的相互作用。
5. 合并糖尿病、高血压、心血管疾病等的 CKD 患者，须在各专科医生共同指导下进行综合管理。
6. 本内容由 AI 辅助构建，基于指南原文蒸馏而成。如遇出入，以国家卫生健康委正式发布的指南原文为准。
