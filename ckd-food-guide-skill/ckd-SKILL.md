# 营养学 | 慢性肾脏病食养助手 (CKD Food Guide)

> 基于《成人慢性肾脏病食养指南（2024年版）》| 作者：王润圆

## 📖 简介

本 Skill 是国家卫生健康委发布的**《成人慢性肾脏病食养指南（2024年版）》**的完整蒸馏版，专为成人慢性肾脏病患者（CKD 1-5期）及家属、基层卫生工作者提供科学的膳食指导和中医食养建议。

### 核心特色

- ⚕️ **分期管理**：针对 CKD 1-2期 / 3-5期 / 5期透析阶段 提供差异化的饮食方案
- 🌿 **中医辨证**：覆盖 7 种证型（气虚/血虚/阴虚/阳虚 + 湿/热/瘀）的食养方
- 🍽️ **84 套食谱**：7 地区 × 3 分期 × 4 季节 = 84 套全天膳食食谱
- 📊 **数据速查**：常见食物钠/钾/磷/蛋白质含量表 + 食物交换表
- ⚠️ **电解质安全**：钾/磷/钠个体化管理，高钾血症预警

### 适用人群

| 人群 | 说明 |
|------|------|
| CKD 1-2期患者 | 以植物性食物为主，正常蛋白饮食 |
| CKD 3-5期非透析 | 低蛋白饮食（0.6g/kg/d），主食优化 |
| CKD 5期透析 | 高蛋白饮食（1.0-1.2g/kg/d），仍控电解质 |
| 家属照护者 | 实操烹饪指导 + 食谱参考 |
| 基层卫生工作者 | 营养管理工具 + 患者教育材料 |

## 🚨 使用前必读

>

## 🛡️ 安全声明

**适用范围**：本Skill适用于成人慢性肾脏病（CKD 1-5期及透析期）人群的辅助营养管理。

**重要限制**：
- ⚠️ **不适用于**：急性肾损伤、肾移植术后、病情复杂合并多器官功能衰竭者——以上情况须在专科医生指导下进行营养管理
- ⚠️ **不可替代**：本Skill内容不可替代药物治疗、血液透析/腹膜透析治疗及专业医疗诊断
- ⚠️ **CKD 3期以上**：使用营养方前须结合血钾、血磷、血尿酸、血白蛋白、血气分析等指标，在肾内科医生或临床营养师指导下选用
- ⚠️ **透析患者**：蛋白质、钾、磷、钠、水分等限制与非透析期有本质差异，须以透析中心医嘱为准

## 💡 Gotchas（避坑提醒）

1. **低钠盐 = 高钾盐，CKD患者禁用！** 市面上的"低钠盐""健康盐"以氯化钾代替氯化钠，CKD 3期以上摄入可致高钾血症→心脏骤停。

2. **分期不同，限制天差地别。** CKD 1-2期限蛋白宽松（0.8g/kg/d），3-5期限到0.6g/kg/d，透析期反而要高蛋白（1.0-1.2g/kg/d）。不要混用。

3. **隐性磷：加工食品的磷添加剂。** 可乐、方便面、火腿肠、速冻水饺中的磷酸盐添加剂吸收率接近100%（天然食物中的磷只吸收40-60%），是CKD患者高磷血症的主因。

4. **高钾水果陷阱。** 香蕉、橙子、猕猴桃、芒果、樱桃、牛油果、椰子水——CKD 3期以上高钾风险者须严格控制，不是"多吃水果有益"。

5. **中药≠安全。** 含马兜铃酸、雷公藤、关木通等成分的中药有明确肾毒性；即使是"补药"，未经肾内科评估不可自行服用。

6. **植物蛋白 vs 动物蛋白。** CKD患者以优质动物蛋白为主（鸡蛋、牛奶、鱼肉），植物蛋白（豆类、坚果）含非必需氨基酸多、磷含量高，需限制。

7. **少尿/无尿期严格限水。** 非透析CKD 3-5期少尿者，每日液体摄入量 = 前一日尿量 + 500ml，不可随意"多喝水排毒"。

8. **过度限制≈营养不良。** 一味"低蛋白、低钾、低磷"可能导致蛋白质-能量消耗（PEW），反而加速肾功能恶化。须在营养师指导下平衡。
