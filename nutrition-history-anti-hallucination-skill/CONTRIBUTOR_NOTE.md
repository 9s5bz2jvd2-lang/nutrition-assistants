# Contributor Note

本仓库整理自一个本地 custom skill，目标是公开复用“营养学历史文献调研防幻觉流程”。维护与贡献时请注意：

## 内容边界

1. 本项目是研究流程与写作防幻觉工具，不是医学建议工具。
2. 不应加入任何“古代食疗可直接证明现代疗效”的表述。
3. 涉及疾病、治疗、药食同源、补充剂或临床营养时，应明确需要现代证据另行核验。
4. 对“第一部”“最早”“首创”等表述必须要求范围限定和可回溯来源。

## 证据纪律

- 搜索片段、百科页面、模型生成内容只能作为线索，不应升级为结论。
- 对古籍与旧文献，应优先补充版本、页码、卷次、校注情况与馆藏/影印来源。
- 对 OCR 文本要保留可能误读项，尤其是人名、书名、异体字和术语。

## 修改 SKILL.md 时

- 保持 front matter 中的 `name` 稳定，避免破坏 skill 触发。
- 新增规则应尽量可操作，最好带模板或判定条件。
- 不要删除“古代食疗不是现代疗效证据”的边界说明。
- 如加入外部链接或示例来源，请确认其公开可访问且不含隐私信息。

## 公开发布前检查

- 不包含 token、cookie、私有路径、个人联系方式或未授权材料。
- README 已说明非医学建议。
- 若需要正式开源，请仓库所有者补充 LICENSE。
