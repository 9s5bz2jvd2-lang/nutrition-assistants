# 骨质疏松症食养指南 食谱概览

## 食谱总览

本指南提供 **7个地区 × 4个季节 × 3个能量级别 = 84套全天膳食食谱**

### 能量级别（骨质疏松症特有）
- **食谱1**：约1600kcal（轻体力活动/减重期）
- **食谱2**：约1900kcal（一般活动量）
- **食谱3**：约2100kcal（中体力活动/恢复期）

### 食谱特点
- 每套食谱均标注蛋白质、碳水化合物、脂肪克数及占总能量比
- ⭐**含钙含量标注**（骨质疏松指南特色！每套食谱均标注钙mg数）
- 含食药物质标注（*标记），方便中医证型匹配
- 油盐用量明确：植物油≤25g/d，盐<5g/d
- 优质蛋白比例≥50%

### 使用方法
1. 根据所在地区选择对应食谱
2. 根据季节选择当季食谱
3. 根据能量需求选择合适能量级别
4. 同一季节内不同能量级别食谱可参照食物交换表调整
5. 结合自身中医证型，参考附录5食养方搭配使用

## 示例食谱

### 东北地区 春季食谱1（总能量约1600kcal）

| 餐次 | 食谱 |
|------|------|
| 早餐 | 莜麦馒头（面粉 30g，莜麦 20g）；蒸芋头（芋头 50g）；煮鸡蛋（鸡蛋 50g）；低脂牛奶（300mL） |
| 茶饮 | 黄芪八角茴香茶（黄芪*3g，八角茴香*1g） |
| 午餐 | 二米饭（大米 50g，小米 20g）；清蒸鲈鱼（鲈鱼 80g）；白菜炖豆腐（白菜 150g，北豆腐 100g）；蒜蓉西蓝花（西蓝花 150g） |
| 加餐 | 苹果（200g） |
| 晚餐 | 玉米面发糕（玉米面 50g）；鸡肉丸子冬瓜汤（鸡胸肉 50g，冬瓜 100g）；蒜蓉油麦菜（油麦菜 150g） |
| 油、盐 | 全天总用量：植物油 23g，盐＜5g |

> 注：1. 本食谱提供蛋白质 70g，其中优质蛋白比例为 55%，碳水化合物 217g，脂肪 53g。宏量营养素占总能量比为：蛋白质 18%，碳水化合物 52%，脂肪 30%。钙 1158mg。2. *为食谱中用到的食药物质，如黄芪、八角茴香。
