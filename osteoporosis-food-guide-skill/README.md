## 指南出处

- **全称**：《成人骨质疏松症食养指南（2026年版）》
- **发布单位**：国家卫生健康委办公厅
- **制定依据**：《健康中国行动（2019—2030年）》《国民营养计划（2017—2030年）》《全民健康生活方式行动方案（2017—2025年）》"三减三健"专项行动

## 功能特点

- **骨骼风险评估**：IOF一分钟自测19问 + OSTA指数计算（⭐骨质疏松特色）
- **钙与维生素D**：精准补钙方案，VD"搬运工"原理，25(OH)D目标管理
- **食养原则**：8条官方原则的详细解读与实操建议
- **中医食养**：2种证型（脾肾阳虚/肝肾阴虚）+ 四季养生
- **地区食谱**：7大地区×4季节×3能量等级完整食谱，含钙含量标注
- **防跌倒指导**：运动方案+日晒方案+体重管理
- **科普风格**：说人话、给具象量、指误区——精准但不端着

## 关键数据速查

| 项目 | 推荐量 | 通俗理解 |
|------|--------|---------|
| 钙（50岁以上） | 1000-1200mg/天 | 一斤奶≈500mg，还得再加 |
| 维生素D（18-64岁） | 10μg(400IU)/天 | 一粒标准VD片 |
| 维生素D（≥65岁） | 15μg(600IU)/天 | 老人要加量 |
| 蛋白质 | 1.2-1.5g/kg/天 | 60kg要72-90g |
| 奶制品 | ≥300mL/天 | 一杯奶打底 |
| 食盐 | <5g/天 | 一个啤酒瓶盖 |
| 烹调油 | ≤25g/天 | 两汤匙封顶 |
| 日晒 | 10-20分钟/天 | 露手臂面部，避开正午 |
