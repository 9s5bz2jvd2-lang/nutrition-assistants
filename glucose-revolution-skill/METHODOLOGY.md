# TKP → Skill 转化方法论

## 概述

本文档解释如何将《控糖革命》的 80 个 TKP (Teaching Knowledge Point) 转化为可用的 AI Skill。

## 关键设计决策

### 1. 保留 TKP 结构

**决策**：在系统提示词中保留 TKP 编号引用

**原因**：
- 可追溯性：用户可查阅原始 TKP 获取更详细内容
- 结构化：TKP 编号本身就是知识索引
- 可扩展：未来可直接链接到 TKP 数据库

### 2. 三级教学法适配

**TKP 原始结构**：
```
初学者解释 → 进阶者解释 → 常见误解
```

**Skill 适配**：
```
自动检测用户水平 → 选择对应层级 → 必要时切换
```

**实现方式**：
- 在系统提示词中明确定义三级特征
- 使用 "根据用户水平自动适配" 指令
- 提供水平切换的显式指令

### 3. VA (Verified Assertion) 模式

**来源**：impersonate-meta 方法论

**适配**：
```
[VA-1: 核心断言] → 一句话点明
[VA-2: 机制解释] → 三选一教学
[VA-3: 实用建议] → 可操作建议
[VA-4: 常见误区] → 避坑指南
[REF: 来源] → TKP 引用
```

### 4. 症状-机制映射

**TKP 数据**：
- TKP051-055: 短期症状（困倦、饿怒、睡眠、头痛、脑雾）
- TKP056-060: 长期症状（皮肤、衰老、阿尔茨海默、癌症、心脏病、PCOS、糖尿病、脂肪肝）

**Skill 转化**：
创建一个症状→机制→建议的快速映射表，便于：
- 用户描述症状时快速关联
- 提供针对性建议
- 解释"为什么会这样"

### 5. 10个窍门的优先级

**TKP 数据**：每个窍门有2个TKP（原理+实践）= 16个TKP

**Skill 转化**：
- 创建窍门速查表
- 定义组合策略（基础/强效/全效）
- 提供实施优先级建议

## 四维配置详解

### 维度1：知识基础

**内容来源**：
- 80个TKP的"核心概念"字段
- 三条知识链的图形化表示
- 评估题库的常见问题

**Skill 实现**：
```
系统提示词中的 "## 四维配置" 部分
├── 三大知识模块
├── 三条核心因果链
└── 10个控糖窍门速查表
```

### 维度2：教学风格

**内容来源**：
- TKP的"教学解释"三字段
- "学习活动"的分级设计

**Skill 实现**：
```
三级教学法表格
├── 初学者（类比为主）
├── 进阶者（机制解释）
└── 专家级（研究引用）
```

### 维度3：回应模式

**内容来源**：
- TKP的7字段结构
- impersonate-meta的VA模式

**Skill 实现**：
```
标准回应结构
├── VA-1: 核心断言
├── VA-2: 机制解释
├── VA-3: 实用建议
├── VA-4: 常见误区
└── REF: 来源标注
```

### 维度4：人格特质

**内容来源**：
- 原书的写作风格（科学+亲和+实用）
- 用户记忆中的"老头子"风格偏好

**Skill 实现**：
```
声音特质
├── 专业但不居高临下
├── 鼓励但不盲目乐观
├── 直接但不生硬
└── 实用但不琐碎
```

## 跨平台适配

### WorkBuddy
- 安装路径：`~/.workbuddy/skills/custom/`
- 配置文件：`skill.yaml`
- 触发方式：关键词触发

### 灵台 TUI
- 安装路径：`~/.lingtai/custom/`
- 兼容 WorkBuddy 格式
- 可能需要额外配置 `manifest.json`

### 通用 AI 平台
- 可直接使用 `system_prompt.md` 作为系统提示词
- 手动复制粘贴即可使用

## 使用建议

### 对学习者
1. 从 `QUICK_REFERENCE.md` 开始
2. 使用 Skill 进行日常咨询
3. 深入学习时查阅具体 TKP

### 对教师/教练
1. 使用 Skill 设计课程内容
2. 参考 80 个 TKP 构建教学大纲
3. 使用评估题库检验学习效果

### 对开发者
1. 修改 `system_prompt.md` 调整教学风格
2. 扩展 `skill.yaml` 添加新功能
3. 基于 TKP 结构开发交互式课程

## 局限与未来

### 当前局限
1. 无法直接访问 TKP 文件内容（需手动复制）
2. 缺乏用户进度追踪
3. 无多媒体支持

### 未来扩展
1. 链接到 TKP 数据库
2. 添加 CGM 数据解读功能
3. 开发个性化方案生成器
4. 构建社区分享功能
