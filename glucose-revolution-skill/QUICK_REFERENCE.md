## 5分钟上手：最重要的3个窍门

### 窍门1：改变饮食顺序 ⭐⭐⭐
```
正确顺序：蔬菜 → 蛋白质/脂肪 → 淀粉/糖

实际应用：
- 聚餐时先吃凉菜
- 在家吃饭时先吃蔬菜
- 吃盖浇饭时先把菜吃完，最后吃米饭
```

### 窍门2：餐前绿色开胃菜 ⭐⭐
```
每餐前吃一小份生蔬菜（约1拳头）

推荐选择：
- 生菜、黄瓜、番茄
- 凉拌菠菜
- 一小碗蔬菜沙拉
```

### 窍门3：餐前喝醋 ⭐⭐
```
餐前10分钟喝1-2勺醋（用水稀释）

可选：
- 苹果醋
- 陈醋
- 柠檬汁
```

## 血糖峰值自测

以下症状可能表明你有血糖波动问题：

| 症状 | 频率 |
|------|------|
| □ 餐后困倦 | 每天/经常/偶尔 |
| □ 饿怒症（饿的时候情绪暴躁） | 每天/经常/偶尔 |
| □ 对甜食有强烈渴望 | 每天/经常/偶尔 |
| □ 减肥困难 | 是/否 |
| □ 皮肤痤疮或炎症 | 是/否 |
| □ 注意力涣散/脑雾 | 每天/经常/偶尔 |
| □ 经期不规律 | 是/否 |

**结果解读**：
- 命中4项以上 → 强烈建议开始控糖
- 命中2-3项 → 值得关注，建议调整
- 命中0-1项 → 继续保持，预防为主

## 果糖冰山模型 🧊

```
           ↑ 血糖仪看到的（葡萄糖）
          /|\
         / | \   血糖峰值
        /  |  \  
       /___|___\________ 水面
      /            \
     /              \    血糖仪看不到的（果糖）
    /                \   果糖峰值 → 肝脏脂肪
   /                  \  → 10倍糖化 → 尿酸
  /____________________\
```

**关键认知**：
- 蔗糖 = 葡萄糖（50%）+ 果糖（50%）
- 血糖仪只能检测葡萄糖
- 果糖虽然"隐形"，但危害可能更大

## 一日控糖示范

### 早餐
- ✅ 鸡蛋2个 + 牛油果半个 + 少量全麦面包
- ❌ 避免：白粥、油条、甜豆浆、果汁

### 午餐
1. 先吃：凉拌黄瓜/沙拉
2. 再吃：肉类/豆腐 + 蔬菜
3. 最后：米饭（如有）
- 餐前：1勺苹果醋稀释饮用

### 下午茶
- ✅ 坚果一小把 / 芝士一块 / 黑咖啡
- ❌ 避免：饼干、蛋糕、含糖饮料

### 晚餐
- 同午餐原则
- 饭后：散步10-20分钟

### 甜点（如想吃）
- ✅ 放在正餐后吃
- ❌ 不要：作为零食单独吃

## 常见误区纠正

| 误区 | 真相 |
|------|------|
| "我吃得不多，所以血糖没问题" | 少量高GI食物仍会造成峰值，关键是进食顺序 |
| "水果很健康，可以随便吃" | 水果含果糖，过量仍会造成果糖峰值 |
| "糙米比白米健康得多" | 糙米只是略好，关键是"什么时候吃"和"跟什么一起吃" |
| "少食多餐可以控制血糖" | 频繁进食会让胰岛素持续高位，反而阻碍脂肪燃烧 |
| "代糖可以随意使用" | 代糖虽无热量，但可能影响肠道和胰岛素敏感性 |

## 三种知识链

### 为什么生病？
```
血糖峰值 → 自由基/糖化/胰岛素过多 → 慢性炎症 → 慢性病
```

### 为什么控糖能减肥？
```
血糖平稳 → 胰岛素稳定 → 脂肪燃烧通道打开 → 体重下降
```

### 为什么需要纤维？
```
纤维 → 减缓胃排空 → 减缓葡萄糖吸收 → 血糖平稳
```

## 进阶：窍门组合

**基础组合**（降低30-40%峰值）：
- 饮食顺序 + 绿色开胃菜

**强效组合**（降低50-60%峰值）：
- 基础组合 + 餐前醋 + 饭后运动

**全效组合**（降低60-75%峰值）：
- 6个窍门同时使用
