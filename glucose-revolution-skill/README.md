## 🎯 能力清单

### 症状-对策速查

| 你的困扰 | 我的回应 |
|---------|---------|
| 餐后秒睡 | 饮食顺序 + 餐前醋 + 饭后溜达 |
| 减肥困难 | 胰岛素机制 + 平稳早餐 + 停止少食多餐 |
| 皮肤作妖 | 戒糖饮料 + 饮食顺序 + Omega-3 |
| 饿怒症 | 平稳早餐 + 咸香零食替代 |
| 脑雾迷糊 | 饮食顺序 + 避免血糖过山车 |

### 场景化指南

- **聚餐/应酬** — 先吃菜、喝酒前垫肚子、干型葡萄酒搞起
- **便利店** — 茶叶蛋+蔬菜沙拉、烤鸡腿+关东煮
- **想吃甜食** — 放在正餐后吃、先吃蛋白质缓冲
- **早餐设计** — 鸡蛋+牛油果+少量全麦，告别粥+包子

### 10个控糖小窍门

1. ✅ 正确的饮食顺序（先菜后肉最后饭）
2. ✅ 增加绿色开胃菜（餐前塞点生的）
3. ✅ 停止计算热量（代谢比热量重要）
4. ✅ 平稳早餐原则（高蛋白低精制碳水）
5. ✅ 所有糖都一样（别被"健康糖"忽悠）
6. ✅ 集中吃甜点（当饭后甜点，别当零食）
7. ✅ 餐前喝醋（酸是血糖的刹车）
8. ✅ 饭后运动（肌肉帮你吃糖）
9. ✅ 咸香零食替代（坚果芝士比饼干强）
10. ✅ 给碳水"穿上外衣"（蛋白质包裹碳水）

## 💬 对话示例

### 示例1：自测评估
```
你：我想做个自测

Skill：来，做个30秒的快问快答 👀
      以下症状，你中了几条？
      1. 饭后秒变睡神
      2. 饿怒症晚期（饿了看谁都不顺眼）
      3. 甜食成瘾
      4. 减肥困难户
      ...

你：1, 3, 4

Skill：好家伙，你这不是血糖波动，是血糖在"蹦迪"啊 💃
      但别慌！好消息是：这些都是吃出来的，也能吃回去
      咱从明天开始调整，一周后告诉我效果！
```

### 示例2：减肥咨询
```
你：想减肥但少吃多动也不瘦

Skill：姐妹/兄弟，你可能被"少吃多动"忽悠了 😅
      【真相】不是你不够努力，是胰岛素在搞事情
      【骚操作】停止少食多餐，早餐直接上鸡蛋+牛油果
      【原理】血糖稳了，胰岛素降了，脂肪自然就烧了

你：早餐吃什么？

Skill：告别粥+包子！试试：
      2个鸡蛋 + 半个牛油果 + 少量全麦面包
      到中午都不饿，还不会想吃零食
```

### 示例3：场景咨询
```
你：今晚有聚餐，怎么办？

Skill：聚餐也能控糖？必须能！而且不用当苦行僧 🎉
      1. 开局先点凉拌菜（假装养生，实则控糖）
      2. 喝酒前垫点肉（保护胃，还能挡酒劲）
      3. 主食最后吃，意思意思就行
      4. 选干型葡萄酒，甜酒say no
      
      记住：咱是去享受的，不是去受罪的！
```
