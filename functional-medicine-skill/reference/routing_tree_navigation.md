# 使用方法：树状渐进式披露

## 核心原则

不要一次性加载全部七层内容。根据用户的具体问题，从根节点出发，沿匹配分支逐步深入，只加载回答当前问题所需的知识层。每个节点展开到足够回答问题的深度即止。

## 树状导航结构

```
根节点：用户输入
│
├─ 理论问题分支
│   ├─ 概念/定义 → 第一层（快速参考）
│   ├─ 某个失衡系统的机制/检测/干预 → 第二层对应小节
│   │   └─ 需要更细节 → 第七层对应章节原文
│   ├─ 某个营养素的作用机制 → 第三层营养素详解
│   │   └─ 需要更细节 → 第七层 ch11_nutrition.txt 或对应章节
│   ├─ 基因多态性 → 第三层基因组学
│   ├─ 病例分析 → 第五层病例库
│   │   └─ 需要完整病例细节 → 第七层 ch17_cases.txt
│   └─ 特殊人群（儿童/抗衰老/慢性疲劳） → 第六层
│
├─ 配方设计分支
│   ├─ Step 1 信息采集
│   │   ├─ 信息充分 → 继续
│   │   └─ 信息不足 → 追问2-3轮关键信息（主要症状、病史、生活方式、用药史）
│   ├─ Step 2 失衡识别
│   │   ├─ 匹配第二层失衡信号表
│   │   ├─ 排列主次失衡（找最上游的失衡）
│   │   └─ 如有基因检测 → 查第三层基因调整表
│   ├─ Step 3 营养素选择
│   │   ├─ 3a 教材推荐：从第四层营养素选择表匹配对应失衡系统
│   │   ├─ 3b 文献发现（强制）：搜索目标适应症领域所有营养素RCT证据，
│   │   │   不限于教材推荐的成分，发现教材未覆盖的有效营养素
│   │   ├─ 3c 合并清单：教材推荐 + 文献发现的营养素合并为候选清单
│   │   ├─ 3d 筛选决策：对候选清单逐项评估证据等级、剂量安全性、
│   │   │   与用户情况的适配度，确定最终配方成分
│   │   ├─ 应用协同原则（不单用一种）
│   │   └─ 如有基因多态性 → 应用基因调整表
│   ├─ Step 4 分阶段处方
│   │   ├─ 按处方格式模板输出（强化期+维持期）
│   │   ├─ 生活方式配合建议
│   │   ├─ 建议检测项目
│   │   └─ 注意事项
│   ├─ Step 5 循证证据表+各国标准（强制，配方的固定组成部分）
│   │   ├─ 从配方中提取所有营养素+剂量
│   │   ├─ 对每个营养素三轮搜索：人群RCT、动物实验、分子通路
│   │   ├─ 查询各国使用标准（中国RNI/UL、美国RDA/UL、欧盟EFSA）
│   │   ├─ 输出表1（循证证据：RCT+动物实验+分子通路）
│   │   ├─ 输出表2（各国使用标准与法规参考）
│   │   ├─ 每个营养素都要列，查到什么写什么，不编造
│   │   └─ 如证据与教材推荐冲突 → 标注并给出调整建议
│   └─ Step 6 局限性声明
│
└─ 混合分支（理论+配方）
    └─ 先走理论分支解释机制，再走配方分支输出方案
```

## 树状导航执行规则

1. 从用户输入中提取关键词和健康问题
2. 在树中匹配最相关的分支
3. 只展开到回答问题所需的深度——不要过度加载
4. 如果当前深度不够回答，向下一层深入
5. 如果用户问题跨多个分支，分别走各分支后合并
6. 配方设计分支必须走完Step 1-6全部步骤，其中Step 3b文献发现为强制环节，不可跳过
7. RCT查询在配方生成后自动触发，不等用户要求
8. Step 0文献发现的搜索轮次视领域文献量而定，至少3轮，确保覆盖教材未收录的有效成分

## 深度研究模式

当用户要求详细解释某个营养素作用机制或某个病例配方逻辑时，沿树状分支深入到第七层「完整知识库」中的对应章节原文。所有教材内容已内嵌于本skill中。

第七层文件索引（按主题快速定位）：
- 基础理论/诊疗路径 → ch01_overview, ch09_homeostasis, ch10_genomics
- 消化/肠道 → ch02_digestion
- 免疫/炎症 → ch03_immune
- 解毒/生物转化 → ch04_detox
- 激素/HPA轴/胰岛素/甲状腺/性激素 → ch05_hormone
- 氧化还原/线粒体 → ch06_oxidative
- 循环/血脂/心血管 → ch07_circulation
- 结构/骨骼 → ch08_structural
- 营养素详解 → ch11_nutrition
- 抗衰老 → ch12_antiaging
- 慢性代谢病 → ch13_metabolic
- 女性激素 → ch14_female
- 儿童4-A病 → ch15_children4A
- 其他慢病 → ch16_other
- 病例分享 → ch17_cases
- 营养达人荟版全书 → fm2_ch01~fm2_ch16（其中 fm2_ch16_supplements 为营养素补充剂专章，1147KB）
- 按主题蒸馏的摘要 → distilled_01~07（基础理论、消化、免疫解毒、激素代谢、氧化循环、基因营养、病例特殊人群）
