# 第一层：快速参考

## 功能医学定义

功能医学是寻找病因的医学。它关注"人为什么生病"而非"得了什么病"，从遗传、环境、心理、生活方式和生理的联动关系入手，研究人体由生理功能下降到病理改变的发病过程。功能医学是"上游医学"——追溯患者病史和生活史，回到机体开始出现紊乱的那一刻。功能医学也是"治未病医学"，通过维持机体基本生理过程保障器官功能平衡。

## 六项原则

1. 生化个体差异性——每个人的生理生化系统由遗传与环境共同塑造
2. 以病人为中心而非以疾病为中心——一种失衡可导致多种疾病，一种疾病可源于不同失衡
3. 生理、心理和精神是内外动态平衡的有机整体
4. 人体内部是连接各生理要素的互联网
5. 没有临床疾病不等于健康——健康是器官功能的最佳状态
6. 提高器官储备水平带来健康并增加寿命

## 七大失衡

1. 消化吸收与肠道菌群失衡——肠漏、菌群失调、消化酶不足
2. 免疫失衡与炎症——慢性炎症、自身免疫、过敏
3. 解毒与生物转化失衡——肝脏解毒能力下降、毒素累积
4. 激素与神经递质失衡——内分泌紊乱、情绪障碍
5. 氧化还原失衡与线粒体损伤——自由基过多、能量代谢障碍
6. 循环运输失衡——血脂异常、内皮功能紊乱、微循环障碍
7. 结构性失衡——脊柱、关节、细胞膜结构异常

这些失衡不是独立的，它们形成链条：消化失衡→免疫失衡→炎症加重→激素紊乱→氧化应激→循环障碍→结构损伤。

## Go-to-it诊疗路径

Gather（收集信息）→ Organize（组织整理，放入时间轴和矩阵图）→ Tell（向患者重述病史，挖掘前置/诱发/介质因素）→ Order（优先排序，找出不足/过剩模型）→ Initiate（开始治疗）→ Track（跟踪随访评估）。

## 干预五个层面

1. 整体机体（身心灵交互）：睡眠、运动、压力、饮食对其他系统的连锁影响
2. 器官系统：针对特定器官的功能支持（肠道、肝脏等）
3. 细胞代谢：辅酶（NADH/NADPH）、维生素、矿物质辅因子；MTHFR多态性影响叶酸代谢
4. 亚细胞/线粒体：ω-3脂肪酸修复膜、肉碱转运脂肪酸、辅酶Q10电子传递
5. 基因表达：NFκB通路、炎症介质、DNA修复（叶酸/B12）、表观遗传调控

## 功能医学vs传统医学

功能医学针对机体功能状态，目标是恢复功能平衡和预防疾病，全方位干预且生活方式首位，疗程长需终身坚持，患者主动参与。临床医学针对疾病病症，目标是管理疾病，以药物为最终手段，疗程短，患者被动接受。功能医学检测探究"为什么生病"（基因、细胞营养水平、肠道菌群、抗氧化水平、激素代谢物等），临床医学检测为诊断"是什么病"。

## 核心概念速查

- 功能医学矩阵：按失衡维度×生活方式因素组织的诊断工具
- 功能医学时间轴：追溯疾病起源的时间线
- 器官储备：心、脑、肺、肝、肾等重要器官在遭遇伤害时的储备功率
- 生化个体差异性：DNA+环境共同决定的个体特性
- 内环境稳态：机体动态平衡，功能医学的核心目标
- 前置因素（antecedents）：基因易感性、母体子宫内获得、营养、毒素暴露、创伤
- 诱发因素（triggers）：创伤、微生物、过敏原、环境毒素、压力生活事件
- 介质因素（mediators）：激素、神经递质、细胞因子、自由基、代谢产物
- PTSD法则：Production（合成）→ Transport（转运转化）→ Sensitivity（敏感性）→ Detoxification（降解排出）
- 4-A综合征：自闭症、多动症、哮喘、过敏症
- 5R肠道修复：Remove（去除）→ Replace（补充）→ Reinoculate（再接种）→ Repair（修复）→ Rebalance（重建平衡）
