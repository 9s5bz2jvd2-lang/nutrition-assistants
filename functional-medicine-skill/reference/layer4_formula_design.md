# 第四层：补剂配方设计

## 配方设计核心流程

```
信息采集 → 失衡识别（主次排列）→ 五层面干预定位 → 营养素选择 → 分阶段处方 → 注意事项
```

### 第一步：信息采集

必需信息：主要健康问题/当前症状、既往病史与用药史、生活方式（饮食、睡眠、排便、运动、压力）。

增强信息（有则更好）：体检阳性指标（血糖、血脂、肝肾功能、HbA1C等）、功能医学检测（食物不耐受、激素水平、重金属、肠道菌群等）、基因检测（MTHFR、GSTT1、COMT、CYP1B1、SOD2、NQO1等）、家族疾病史。

信息不足时：基于已有信息给出方向性建议，并明确标注"建议检测以下项目以精确定位"。

### 第二步：失衡识别

| 失衡系统 | 关键提示信号 |
|---------|------------|
| 消化吸收与菌群失衡 | 排便异常、腹胀、食物不耐受、偏食、抗生素史、肠镜异常 |
| 免疫与炎症失衡 | 过敏、湿疹、鼻炎、反复感冒、自身免疫病、CRP/ESR升高 |
| 解毒与生物转化失衡 | 重金属超标、化学敏感、GSTT1缺失、环境毒素暴露 |
| 激素与神经递质失衡 | 月经紊乱、性欲低下、失眠焦虑、甲状腺异常、胰岛素抵抗 |
| 氧化还原失衡与线粒体损伤 | 慢性疲劳、运动耐力差、SOD2突变、辅酶Q10低 |
| 循环运输失衡 | 血脂异常、高血压、同型半胱氨酸升高、微循环障碍 |
| 结构性失衡 | 骨密度低、关节疼痛、细胞膜流动性差（胰岛素抵抗） |

排列原则：找到首要失衡（通常1-2个），以此为切入点，同时兼顾协同失衡。功能医学的精髓是"找到最上游的那个失衡"。

### 第三步：五层面干预定位

层面一：整体机体（身心灵交互）— 生活方式调整为主
层面二：器官系统 — 肠道（益生菌、谷氨酰胺、消化酶）、肝脏（肝脏解毒胶囊、NAC）
层面三：细胞代谢 — 氨基酸、脂类、辅酶、维生素、矿物质；MTHFR突变→活性叶酸
层面四：亚细胞/线粒体 — ω-3修复膜、辅酶Q10、左旋肉碱、α-硫辛酸、谷胱甘肽
层面五：基因表达 — EPA/DHA抑制NFκB、叶酸B12促DNA修复、姜黄素激活Nrf2

### 第五步：分阶段处方

第一阶段：强化期（前1-3个月）— 益生菌剂量偏高（600亿CFU/日），营养素种类偏全（同时覆盖2-3个失衡系统），重点解决首要失衡，抗炎/抗氧化剂量偏高。

第二阶段：维持期（3个月后）— 益生菌减量至200-300亿CFU/日，营养素种类精简，维持核心营养素，根据复查指标调整。维生素D维持期不减量（因普遍缺乏）。

## 基因多态性调整表

| 基因 | 变异类型 | 营养素调整建议 |
|------|---------|-------------|
| MTHFR C677T | 杂合（+/−） | 叶酸改为5-MTHF活性形式，400μg/日 |
| MTHFR C677T | 纯合（+/+） | 5-MTHF 800μg/日 + 甜菜碱1000mg/日 |
| GSTT1 | 缺失 | 加强谷胱甘肽支持：NAC 1200mg/日 + 谷胱甘肽500mg/日 |
| SOD2 | 纯合 | 加强线粒体抗氧化：辅酶Q10 200mg + α-硫辛酸600mg |
| COMT | 杂合 | 甲基化营养素剂量减半（过量甲基化致焦虑） |
| CYP1B1 | 纯合 | 加强雌激素代谢支持：DIM/I3C + 钙D葡萄糖酸 |
| NQO1 | 纯合 | 辅酶Q10必须用泛醇（ubiquinol）形式 |
| ApoE | ++ | 严格低脂饮食，鱼油剂量加大至3g/日 |

## 协同原则

抗氧化网络协同：谷胱甘肽↔维生素C↔维生素E↔辅酶Q10↔α-硫辛酸。联合小剂量使用优于单一大剂量。

甲基化协同：5-MTHF + 甲基化B12 + 甜菜碱（TMG）+ B6。四者协同覆盖甲基化与转硫两条途径，不可偏废。MTHFR突变者尤其需要甜菜碱作为旁路甲基供体。

肠道修复协同：益生菌 + 益生元 + 谷氨酰胺 + 消化酶。益生菌定植需要益生元作为底物，肠黏膜修复需要谷氨酰胺提供能量。

胰岛素敏感性协同：铬 + α-硫辛酸 + ω-3脂肪酸。铬增强胰岛素受体信号，α-硫辛酸改善氧化应激，ω-3修复细胞膜流动性。

解毒协同：NAC + 谷胱甘肽 + 肝脏解毒胶囊 + 姜黄素。NAC是谷胱甘肽前体，姜黄素激活Nrf2解毒通路。
