### KPK-16 疾病定义、分期与流行病学

**核心知识**：高尿酸血症与痛风的定义、四期分类及中国流行病学数据
**适用人群**：成人高尿酸血症/痛风患者、基层卫生工作者、营养指导人员

**食养建议**：
1. **定义与诊断**
   - **高尿酸血症**：嘌呤代谢紊乱引起的代谢性疾病。正常膳食状态下，非同日2次检测空腹血尿酸水平>420μmol/L即可诊断。
   - **痛风**：代谢性疾病，以高尿酸血症和尿酸盐晶体的沉淀和组织沉积为特征。尿酸盐结晶沉积于关节、软组织和肾脏，引起关节炎、肾脏损害等，临床主要表现为反复发作的急性关节炎等。
   - 高尿酸血症和痛风是慢性肾病、高血压、心脑血管疾病及糖尿病等疾病的独立危险因素；长期可导致动脉粥样硬化，增加心血管疾病风险。

2. **疾病分期（4期）**
   - **无症状高尿酸血症期**：无症状高尿酸血症及无症状单钠尿酸盐晶体沉积。
   - **急性痛风性关节炎期**：关节炎突然发作时期，关节红肿热痛，疼痛剧烈。
   - **痛风间歇期**：两次急性痛风性关节炎发作之间的阶段。
   - **慢性痛风性关节炎期**：关节持续疼痛，血尿酸水平持续波动，可伴有痛风石出现。

3. **流行病学数据（2018-2019年中国慢性病及危险因素监测）**
   - 高尿酸血症患病率：**14%**
   - 痛风患病率：**0.86%～2.20%**
   - 性别差异：男性高于女性
   - 城乡差异：城市高于农村
   - 地区差异：沿海高于内陆
   - 趋势：痛风患病率呈逐年上升趋势，发病年龄趋于年轻化

4. **危险因素**
   - 长期摄入高能量食品
   - 大量酒精和（或）高果糖饮料
   - 肥胖是独立危险因素

5. **指南定位**
   - 主要面向未合并肾功能不全等相关合并症的成人高尿酸血症、痛风人群以及基层卫生工作者（包括营养指导人员）
   - **不可代替药物治疗**
   - 有其他合并症者可参考，但需在医生或营养指导人员指导下设计个性化方案

**食材/方剂**：（本知识点为疾病背景知识，具体食材方剂见后续食养原则类KPK）

**注意事项**：
- 诊断需满足"正常膳食状态"+"非同日2次检测"两个条件
- 血尿酸>420μmol/L为诊断阈值，不分性别统一标准
- 食养不能替代药物，合并肾功能不全等其他疾病时须遵医嘱

**关联KPK**：KPK-17（中医辨证分型）、KPK-18（指南使用方法）

### KPK-17 中医认识与辨证分型

**核心知识**：痛风的中医病因病机与四个证型的完整舌脉表现
**适用人群**：需要中医辨证施膳的高尿酸血症/痛风患者

**食养建议**：

1. **中医基本认识**
   - **病性**：本虚标实
   - **病因**：先天禀赋不足 或 后天失养 + 过食肥甘厚味
   - **病机**：日久产生痰浊、湿热、瘀血 → 痹阻于筋骨关节 → 导致疾病发生
   - **发病关键因素**：先天脾胃虚弱或后天饮食失养损伤脾胃 → **脾胃失调、湿浊内生是反复不愈的症结所在**
   - **食养核心**：调理脾胃
   - **推荐调理食材**：白扁豆、玉米须、麦芽、山药、芡实、大枣、橘皮、山楂、五指毛桃、茯苓等

2. **四个证型完整信息**

| 证型 | 常见分期 | 核心表现 | 舌脉 |
|------|----------|----------|------|
| **湿浊证** | 无症状高尿酸血症期 / 痛风间歇期 | 肢体困乏沉重，形体肥胖，嗜食肥甘，口中粘腻不渴，大便黏 | **舌淡胖，或有齿痕，苔白腻，脉滑** |
| **湿热证** | 急性痛风性关节炎期 | 关节红肿灼热疼痛、疼痛剧烈，发作频繁，或伴发热，烦躁不安，口苦、口臭，大便黏滞或臭秽，或大便干 | **舌质红，苔黄腻或黄厚，脉弦滑或滑数** |
| **痰瘀证** | 痛风间歇期 / 慢性痛风性关节炎期 | 关节肿痛，反复发作，局部有硬结或皮色暗红，或关节刺痛，屈伸不灵活、关节变形 | **舌质紫暗，苔白腻，脉弦或弦滑** |
| **脾肾亏虚证** | 慢性痛风性关节炎期 | 关节疼痛反复发作、活动不灵活、僵硬或变形，腰膝酸软，乏力明显，肢体困倦沉重，腹胀，大便黏滞或溏稀 | **舌淡胖，或有齿痕，舌苔白腻，脉沉缓或沉细** |

3. **各证型膳食要点**
   - **湿浊证**：严格控制肥甘厚味（限制嘌呤和蛋白质总摄入量），低盐膳食，避免甜、油腻、酸、涩食物；可食用薏苡仁、橘皮、茯苓代茶饮
   - **湿热证**：严格限制嘌呤、蛋白质摄入，限盐限烟酒，避免外感风寒风热诱发；推荐山竹、西瓜、荸荠，可用赤小豆、木瓜、薏苡仁煮汤
   - **痰瘀证**：多饮水，严格限制高脂、高胆固醇和高盐膳食；可食用木耳、山楂、桃仁等
   - **脾肾亏虚证**：宜食温热性食物，可食生姜、黄芪、茯苓、核桃、荔枝等

**食材/方剂**：
- 湿浊证：橘皮、生姜、赤小豆、郁李仁、茯苓、葛根、菊苣、百合、砂仁、木瓜、黄芪
- 湿热证：薏苡仁、赤小豆、金银花、茯苓、桑叶、葛根、山药、马齿苋、菊苣、荷叶、栀子、决明子、白扁豆、余甘子、百合、鲜白茅根、菊花、五指毛桃、玉米须
- 痰瘀证：橘皮、桔梗、昆布、生姜、佛手、木耳、山楂、桃仁、姜黄、当归、山药、薏苡仁、茯苓、白扁豆、赤小豆、荷叶、葛根、菊苣、莱菔子、紫苏子、莲子、罗汉果、重瓣玫瑰
- 脾肾亏虚证：山药、芡实、莲子、茯苓、白扁豆、赤小豆、薏苡仁、葛根、菊苣、砂仁、大枣、龙眼肉、五指毛桃、人参（人工种植≤5年）、党参、黄芪

**注意事项**：
- 食药物质在限定使用范围和剂量内作为食药物质使用，食用方法请咨询医生或营养指导人员等专业人员
- 新食品原料每种每天食用量不能超过相关规定中使用量，多种联合食用时宜酌情控制
- 已知对某种食药物质过敏者禁用
- 正在服用药物可能与食药物质有禁忌时，需专业人员指导

**关联KPK**：KPK-16（疾病分期对应证型）、KPK-18（三因施膳原则）

### KPK-18 问答版核心要点（Q1-Q9）

**核心知识**：指南制定目的依据原则内容结构及临床使用九大问答要点
**适用人群**：所有需要理解并应用本指南的患者和基层卫生工作者

**食养建议**：

**Q1: 制定目的和意义是什么？**
1. 充分发挥现代营养学和传统食养的中西医各方优势，将食药物质融入合理膳食中，辅助预防和改善高尿酸血症与痛风
2. 针对不同证型人群、不同地区、不同季节编制食谱示例，提出食养指导，提升膳食指导个性化和可操作性
3. 推动中西医结合，传承传统食养文化，切实推进食养服务高质量发展

**Q2: 制定依据是什么？**
- 《健康中国行动（2019—2030年）》——鼓励发展传统食养服务
- 《国民营养计划（2017—2030年）》——发挥中医药特色优势，制定符合我国现状的居民食养指南

**Q3: 制定原则是什么？**
1. **科学性原则**：以营养科学、中医食养理论、食物和健康关系证据的文献研究为依据，参考国内外相关指导原则、指南、标准、典籍等，结合我国实际和专家意见，经反复专家论证
2. **协调一致性原则**：对我国现行食品相关法律、法规、管理规定等进行梳理，确保协调一致
3. **公开透明原则**：广泛征求相关部门以及专家学者、高校、科研院所、医疗机构和社会团体等多方面意见
4. **指导性与实用性相结合原则**：结合不同证型、地区、季节特点列出食物推荐、食谱和食养方示例

**Q4: 国内外相关共识/指南/标准？**

1. 《中国居民膳食指南（2022）》

4. 《高尿酸血症和痛风病证结合诊疗指南（2021-01-20）》

6. 《中国高尿酸血症相关疾病诊疗多学科专家共识（2023年版）》

**Q6: 发布后如何使用？**
- **面向对象**：未合并肾功能不全等其他合并症的成人高尿酸血症、痛风人群，以及基层卫生工作者（包括营养指导人员）
- **患者使用**：结合自身情况合理搭配日常膳食，养成良好饮食习惯
- **基层卫生工作者使用**：结合工作需要及患者实际情况，辅助预防和改善
- **不适用情况**：病情复杂、合并多种疾病者，请在医生或营养指导人员指导下合理膳食
- **重要声明**：不可代替药物治疗

**Q7: 膳食上需要注意什么？**
综合膳食建议汇总：
- **食物多样**：每天≥12种食物，每周≥25种；谷薯类+蔬果+畜禽鱼蛋奶+大豆坚果全覆盖
- **蔬菜充足**：新鲜蔬菜≥500g/d，深色蔬菜占一半以上
- **奶类充足**：≥300mL/d或相当量奶及奶制品
- **足量饮水**：心肾功能正常时2000～3000mL/d，维持尿量>2000mL
- **低嘌呤膳食**：严格控制膳食嘌呤含量
- **限果糖**：限制含糖饮料、鲜榨果汁、果葡糖浆、果脯蜜饯等
- **限酒/不饮酒**：尤其黄酒、啤酒；急性发作/药物控制不佳/慢性期应不饮酒
- **少吃生冷食品**：冰激凌、生冷海鲜等易损伤脾胃功能且增加尿酸盐结晶析出
- **吃动平衡**：保持健康体重，规律作息、劳逸结合，适量运动

**Q8: 中医食养指导原则？**
1. **准确辨证**：根据"因人、因时、因地"三因原则选择相应食药物质
2. **总量纳入**：食药物质所提供的能量与宏量营养素纳入全天饮食总量考虑
3. **过敏禁忌**：已知对某种食药物质过敏者禁用
4. **药物相互作用**：正在服用药物可能与食药物质有禁忌时，应在医生、执业药师及营养指导人员等专业人员指导下食用

**Q9: 参考食谱示例时注意什么？**
1. **调整食材，因人制宜**：根据所在地区、季节选择食谱示例，结合证型、年龄、性别、生理状况调整食物量与品种；有条件可进行专业营养评估和中医辨证
2. **同类互换，享受美食**：利用《常见食物交换表》互换同类食物，注重当地食材选择，兼顾各地饮食文化和口味特点
3. **控制嘌呤，少盐少油**：按推荐量循序渐进降低盐油摄入，培养清淡口味
4. **吃动平衡，健康体重**：注重吃也注重动，保持能量摄入与消耗平衡
5. **养成习惯，贵在坚持**：提升营养健康素养，坚持不懈持续培养健康饮食习惯
