# 痛风营养速查卡 (QUICK REFERENCE)

> 一页速查，无需翻KPK文件即可回答80%常见问题
> 来源：《成人高尿酸血症与痛风营养指南（2024年版）》

---

## 📊 诊断标准

| 项目 | 标准 |
|------|------|
| 高尿酸血症 | 血尿酸 > 420 μmol/L（不分男女） |
| 痛风诊断 | 高尿酸 + 关节红肿热痛 + 关节液检出尿酸盐结晶 |
| 无症状高尿酸 | 尿酸高但从未发作过痛风 |

---

## 🎯 控制目标

| 人群 | 目标血尿酸 |
|------|-----------|
| 无症状高尿酸（首次发现） | < 420 μmol/L |
| 痛风患者（有痛风石/慢性）| < 300 μmol/L |
| 痛风患者（无痛风石） | < 360 μmol/L |

---

## 🔴 嘌呤四级分类

| 等级 | 阈值 (mg/100g) | 建议 | 示例 |
|------|---------------|------|------|
| 🔴 高 | > 150 | ❌ 避免 | 内脏、沙丁鱼、凤尾鱼 |
| 🟠 较高 | 75-150 | ⚠️ 限制 | 红肉、多数鱼类、贝类 |
| 🟢 较低 | 30-75 | ✅ 适量 | 淡水鱼、禽肉、豆制品 |
| 🟢 低 | < 30 | ✅ 放心 | 蔬菜、水果、奶、蛋、谷物 |

## 🍽️ 每日核心指标

| 营养素 | 推荐量 | 通俗理解 |
|--------|--------|----------|
| 💧 饮水 | 2000-3000 mL | 4-6瓶矿泉水 |
| 🧂 食盐 | < 5g/d | 一个啤酒瓶盖 |
| 🛢️ 烹调油 | 25-30g/d | 两汤勺多一点 |
| 🥬 蔬菜 | ≥ 500g/d | 一斤菜，深色占半 |
| 🥛 奶类 | ≥ 300mL/d | 一盒牛奶 |
| 🍎 水果 | 200-350g/d | 两个中等苹果 |
| 🌾 全谷物 | ≥ 主食的30% | 三成粗粮七成细粮 |
| 🏃 运动 | 4-5次/周×30-60min | 隔天动一次 |

## ❌ 饮食"硬禁忌"（无论什么阶段）

1. **所有酒精**：啤酒、白酒、黄酒、红酒（别信"红酒有益"！）
2. **动物内脏**：肝、肾、脑、心
3. **部分海鲜**：沙丁鱼、凤尾鱼、鱼籽、贝类浓汤
4. **浓肉汤/火锅汤**：嘌呤全溶在汤里

---

## 🟢 可以放心吃的（低嘌呤）

鸡蛋、低脂/脱脂牛奶、豆腐/豆浆、绝大多数蔬菜、水果、谷物、土豆、红薯、海参、海蜇

---

## 🧠 常见误区（高价值gotchas）

| # | 误区 | 真相 |
|---|------|------|
| 1 | "不疼了就不用管" | 缓解期才是降尿酸黄金窗口 |
| 2 | "豆腐嘌呤高不能吃" | **植物嘌呤≠动物嘌呤**——豆腐可以吃！ |
| 3 | "海鲜煮汤嘌呤就没了" | 恰恰相反，嘌呤全溶进汤里了 |
| 4 | "喝苏打水能降尿酸" | 苏打水≠碳酸氢钠，效果微乎其微 |
| 5 | "红酒比白酒好，可以喝" | 所有酒精都升高尿酸，没有例外 |
| 6 | "肥胖才得痛风" | 瘦子也会高尿酸，基因是重要因素 |
| 7 | "快速减肥降尿酸" | 过快减肥反而诱发痛风发作 |
| 8 | "维生素C大量吃能降尿酸" | 500mg/d可能微效，但非主要手段 |

## 🌿 中医四证型速查

| 证型 | 关键表现 | 饮食方向 | 推荐食药物质 |
|------|----------|----------|-------------|
| 湿浊内阻 | 身体困重、舌苔白腻 | 化湿降浊 | 薏苡仁、茯苓、白扁豆、陈皮 |
| 湿热蕴结 | 口干苦、舌苔黄腻、关节红肿 | 清热利湿 | 薏苡仁、赤小豆、冬瓜、玉米须 |
| 痰瘀痹阻 | 关节变形、刺痛固定 | 化痰祛瘀 | 山楂、桃仁、丹参 |
| 脾肾亏虚 | 乏力腰酸、反复发作 | 健脾补肾 | 山药、芡实、莲子、枸杞子 |

## 📞 安全红线（就医指征）

- 🚨 关节突发红肿热痛 → **立即就医**
- 🚨 发热寒战伴随关节痛 → 立即就医
- 🚨 肾功能异常者 → 蛋白质和嘌呤需医生指导调整
- 🚨 有痛风石 → 目标尿酸 < 300 μmol/L，需药物
