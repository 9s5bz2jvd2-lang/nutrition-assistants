## 🌿 四大中医证型

| 证型 | 常见阶段 | 核心特征 |
|------|----------|----------|
| **湿浊证** | 无症状期/间歇期 | 困乏肥胖、口中黏腻、舌淡胖苔白腻 |
| **湿热证** | 急性发作期 | 关节红肿灼热痛、口苦口臭、舌红苔黄腻 |
| **痰瘀证** | 间歇期/慢性期 | 关节肿痛反复、有硬结、舌紫暗 |
| **脾肾亏虚证** | 慢性期 | 关节变形、腰膝酸软、乏力、舌淡胖 |
