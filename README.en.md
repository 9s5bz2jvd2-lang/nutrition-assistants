# Nutrition Assistants Collection

English README based only on the existing repository description and visible Chinese documents.

A collection of AI popular-science conversation skills based on National Health Commission dietary-nutrition guides, including assistants for sarcopenia, stroke, osteoporosis, hypertension, hyperlipidemia, hyperuricemia and gout, stunting, diabetes, chronic kidney disease, adult obesity, and childhood/adolescent obesity.

## Included professional workflow skills

- [`yuanjiang-nutritionist-diet-evaluation-assistant-skill/`](yuanjiang-nutritionist-diet-evaluation-assistant-skill/) — Runyuan's nutritionist diet evaluation assistant skill. It helps nutrition professionals structure messy three-day diet records, estimate energy and macronutrient trends with confidence labels, draft follow-up questions, prepare internal judgment material, and generate client-friendly communication wording. It is intended for professional review and does not replace diagnosis, clinical judgment, or individualized treatment prescriptions.
